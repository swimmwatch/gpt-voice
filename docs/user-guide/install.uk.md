# Встановлення, оновлення або видалення GPT-Voice

GPT-Voice має готові до запуску випуски для Windows і Linux. Вам не потрібно окремо встановлювати Node.js, npm,
локальну модель, CUDA або Whisper.

Завантажте пакет для свого комп'ютера з [GitHub Releases](https://github.com/swimmwatch/gpt-voice/releases). Кожен
випуск також містить файли `SHA256SUMS-*.txt` для відповідних платформ. Щоб перевірити завантаження, порівняйте
значення SHA-256, яке показує інструмент перевірки контрольної суми вашої операційної системи, із записом для
завантаженого ресурсу випуску.

> Випуски для macOS призупинені, доки готуються підписання Developer ID і нотаризація. Не встановлюйте неофіційний
> DMG як GPT-Voice: у поточних випусках немає підтримуваного пакета для macOS.

## Виберіть ресурс випуску

| Платформа              | Ресурс випуску           | Використовуйте, якщо                                                            |
| ---------------------- | ------------------------ | ------------------------------------------------------------------------------- |
| Windows                | `GPT-Voice Setup *.exe`  | Вам потрібне звичайне встановлення Windows.                                     |
| Linux сімейства Debian | `gpt-voice_*_amd64.deb`  | Ви використовуєте Ubuntu, Debian, Linux Mint, Pop!_OS або подібний дистрибутив. |
| Linux сімейства RPM    | `gpt-voice-*.x86_64.rpm` | Ви використовуєте Fedora, RHEL, CentOS, openSUSE або подібний дистрибутив.      |
| Linux                  | `GPT-Voice-*.AppImage`   | Вам потрібна портативна збірка без встановлення системного пакета.              |
