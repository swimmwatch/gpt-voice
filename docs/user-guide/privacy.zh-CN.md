# 隐私和数据

GPT-Voice 处理语音、所选文本、凭据和浏览器设置。本页解释了当前的数据路径
以及可用于删除数据的控件。它不会取代您选择的任何服务的隐私政策或条款
使用。

## 数据流

仅当您使用外部服务支持的功能时，GPT-Voice 才会将数据发送到计算机外部：

| 特色                 | 数据已发送                   | 目的地                         |
| -------------------- | ---------------------------- | ------------------------------ |
| 转录 **ChatGPT Web** | 准备好的录音                 | ChatGPT 通过您登录的浏览器会话 |
| 转录 **OpenAI API**  | 准备好的录音和配置的转录选项 | OpenAI 的音频转录端点          |
| **Translate**        | 选定的文本                   | Google Translate               |
| **Prettify**         | 所选文本和您配置的美化提示   | 您配置的 Ollama 或 vLLM 端点   |

使用您信任的帐户和端点，并查看其数据处理条款。本地环回 Ollama 或 vLLM 端点
将请求保留在运行该服务的机器上；远程端点接收文本。使用 HTTPS
非环回美化端点。基于浏览器的服务可以使用[网络设置](settings/network.md)中配置的代理。

GPT-Voice 将成功的转录、翻译和美化结果写入系统剪贴板。经营的
具有剪贴板访问权限的系统和其他应用程序可以保留或读取该值；粘贴后清除或更换
灵敏的输出。

## 本地数据和临时内存

| 数据                | 保存地点和时间                                                                                                |
| ------------------- | ------------------------------------------------------------------------------------------------------------- |
| 成功转录历史        | 本地 SQLite 数据位于 `gpt-voice.sqlite3`，包含请求时间、提供者 ID 和名称以及转录文本。它不存储录制的音频。    |
| 可重试录音          | 在转录失败后，最新准备的音频仅保留在正在运行的应用程序的内存中。开始新的录制或重新启动 GPT-Voice 会将其删除。 |     | 美化结果缓存 | 内存中最多可保存 20 个结果长达 60 秒。缓存上下文取决于所选文本、提示和配置的提供程序设置；当 GPT-Voice 退出时它被删除。 |
| 设置和浏览器身份    | 存储在GPT-Voice的本地应用程序数据中并用于以后的启动。                                                         |
| ChatGPT Web身份验证 | 本地浏览器会话和身份验证数据。使用提供者控件来清除它。                                                        |

选定文本操作在收集选择内容时临时读取并替换剪贴板内容。如果翻译或
美化失败或被取消，GPT-Voice 恢复操作之前捕获的剪贴板值。论成功，
结果保留在剪贴板上供您粘贴。

## 凭证和加密资格

OpenAI API 密钥、vLLM API 密钥和 HTTP/HTTPS 代理密码通过 Electron safe storage 存储
保护可用。 GPT-Voice 不会将这些保存的值返回到其设置视图。如果安全存储是
不可用，GPT-Voice 无法通过该控件保存新机密。

这并不是对应用程序数据目录中的每个文件的一揽子加密声明。特别是转录
历史记录、普通设置和 ChatGPT Web 会话数据都有自己的本地存储行为。不要共享 API 密钥，
代理密码、会话信息、口述文本或包含它们的屏幕截图。

## 删除或重置数据

选择满足您需求的最窄控件：

1. 在“历史记录”窗口中，使用 **Clear history** 永久删除所有保存的转录条目。参见
   [历史记录和托盘](guides/history-and-tray.md)。
2. 在转录提供程序控件中，使用 **Clear session** 表示 ChatGPT Web 或 **Clear API key** 表示 OpenAI API。
   请参阅[提供商设置](settings/providers.md)。
3. 使用[美化设置](settings/prettify.md)中的**Clear API key**控件和密码**Clear**控件
   在[网络设置](settings/network.md)中（如果适用）。
4. 如果系统剪贴板包含您不再需要粘贴的输出，请单独替换或清除系统剪贴板。
5. 要进行完全本地重置，请从托盘中退出 GPT-Voice 并删除其保留的应用程序数据目录：
   Windows 上为 `%APPDATA%\GPT-Voice` 或 Linux 上为 `~/.config/GPT-Voice`。这会删除本地应用程序管理的
   设置、历史记录和保存的提供商数据，并且需要在重新安装或重新启动后重新设置。

删除本地数据是不可逆的。单独卸载应用程序会有意保留这些目录；看到
[安装、更新或删除](install.md)用于特定于平台的卸载行为。
