# Часті запитання

## GPT-Voice транскрибує мовлення самостійно?

Ні. GPT-Voice — це настільна програма, яка надсилає запис вибраному вами постачальнику транскрипції: ChatGPT Web
через сеанс веб-переглядача, у який ви ввійшли, або OpenAI API через ваш власний ключ API. Доступність постачальника, виставлення рахунків,
квоти та умови контролюються цим постачальником. Див. [вибір і керування постачальником транскрипції](guides/providers.md).

## Що залишає мій комп'ютер?

Зупинення запису надсилає підготовлене аудіо до вибраного постачальника транскрипції. Переклад надсилає виділений текст до
Google Translate. Prettify надсилає вибраний текст і його налаштований запит до кінцевої точки Ollama або vLLM. див
[конфіденційність і дані](privacy.md) для повної інформації про потік даних і локальне збереження.

## GPT-Voice вводить результат у мою програму?

Ні. Успішні результати транскрипції, перекладу та Prettify копіюються в системний буфер обміну. Вставте результат
де вам це потрібно. Див. [запис і транскрибування](guides/transcription.md) і
[Перекладіть і покращте виділений текст](guides/text-actions.md).

## Чи можу я використовувати GPT-Voice без ключа OpenAI API?

так ChatGPT Web використовує сеанс браузера ChatGPT замість ключа API. Він окремий від OpenAI API
постачальника та вимог до його облікового запису. Перегляньте [параметри постачальника ](settings/providers.md).

## Чи може GPT-Voice працювати повністю в автономному режимі?

Не для транскрипції чи перекладу: ці функції використовують вибрану дистанційну службу. Prettify може використовувати локальний
Кінцева точка Ollama або vLLM, коли ви запускаєте цю службу на тому самому комп’ютері, але GPT-Voice не встановлює та не керує
кінцева точка для вас. Див. [Параметри Prettify](settings/prettify.md).

## Які платформи підтримуються?

Поточні випуски підтримують Windows і Linux через пакети інсталятора Windows, deb, rpm і AppImage. macOS
звільнення призупинено на час підписання та підготовки нотаріального засвідчення. Див. [встановлення, оновлення або видалення](install.md).

## Оновлення або видалення стирає мої налаштування?

Ні. Звичайні шляхи видалення навмисно зберігають локальні дані програми, включаючи налаштування та збережені дані постачальника.
Скористайтеся інструкціями щодо видалення в розділі [конфіденційність і дані](privacy.md), якщо ви хочете навмисно скинути ці дані.

## Чому мій ярлик або дія виділеного тексту не виконується?

Переконайтеся, що дію ввімкнено, її ярлик збережено, а інша програма не зарезервувала той самий ключ
поєднання. Translation і Prettify запускаються по черзі та чекають, доки закінчиться будь-який запис. див
[налаштування ярликів](settings/shortcuts.md) і [усунення несправностей](troubleshooting.md).

## Чи можу я використовувати проксі?

так GPT-Voice може передавати проксі-сервер HTTP, HTTPS або SOCKS5 у контекст свого браузера. SOCKS5 облікові дані не підтримуються,
і проксі-сервер GeoIP можуть контролювати локалізацію браузера та часовий пояс. Див. [Налаштування мережі](settings/network.md).

## Як очистити стенограму, сеанс або ключ?Використовуйте **Clear history** у вікні «Історія» для збережених транскрипцій. Використовуйте чіткий контроль відповідного постачальника для a

ChatGPT сеанс, OpenAI API ключ, vLLM ключ або пароль проксі. Див. [конфіденційність і дані](privacy.md) для точного обсягу
кожного скидання.
