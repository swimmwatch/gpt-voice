# Усталяванне, абнаўленне або выдаленне GPT-Voice

GPT-Voice мае гатовыя да працы выпускі для Windows і Linux. Вам не трэба асобна ўсталёўваць Node.js, npm, лакальную
мадэль, CUDA або Whisper.

Спампуйце пакет для свайго камп'ютара з [GitHub Releases](https://github.com/swimmwatch/gpt-voice/releases). Кожны
выпуск таксама змяшчае файлы `SHA256SUMS-*.txt` для адпаведных платформаў. Каб праверыць спампаванне, параўнайце
значэнне SHA-256, якое паказвае інструмент кантрольнай сумы вашай аперацыйнай сістэмы, з запісам для спампаванага
актыву выпуску.

> Выпускі для macOS прыпыненыя, пакуль рыхтуюцца падпісанне Developer ID і натарыяльнае засведчанне. Не ўсталёўвайце
> неафіцыйны DMG як GPT-Voice: у бягучых выпусках няма пакета macOS, які падтрымліваецца.

## Абярыце актыў выпуску

| Платформа              | Актыў выпуску            | Калі яго выкарыстоўваць                                                         |
| ---------------------- | ------------------------ | ------------------------------------------------------------------------------- |
| Windows                | `GPT-Voice Setup *.exe`  | Вам патрэбна звычайнае ўсталяванне Windows.                                     |
| Linux сямейства Debian | `gpt-voice_*_amd64.deb`  | Вы карыстаецеся Ubuntu, Debian, Linux Mint, Pop!_OS або падобным дыстрыбутывам. |
| Linux сямейства RPM    | `gpt-voice-*.x86_64.rpm` | Вы карыстаецеся Fedora, RHEL, CentOS, openSUSE або падобным дыстрыбутывам.      |
| Linux                  | `GPT-Voice-*.AppImage`   | Вам патрэбная партатыўная зборка без усталявання сістэмнага пакета.             |
