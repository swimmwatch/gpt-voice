# Настройки сочетаний клавиш

GPT-Voice использует глобальные сочетания клавиш, поэтому они могут работать, когда вы находитесь в другом
приложении. Откройте **Settings** и выберите **Shortcuts**, чтобы просмотреть или изменить их.

| Действие                                        | Сочетание по умолчанию | Когда работает                                                                                                |
| ----------------------------------------------- | ---------------------- | ------------------------------------------------------------------------------------------------------------- |
| **Начать, приостановить или продолжить запись** | `F9`                   | Начинает запись при бездействии, приостанавливает активную запись или продолжает запись с паузы.              |
| **Остановить запись**                           | `F10`                  | Останавливает запись или запись на паузе и начинает расшифровку.                                              |
| **Отмена**                                      | `Escape`               | Отменяет активную запись. Когда активной записи нет, отменяет выполняющийся запрос Prettify.                  |
| **Перевести выделенный текст**                  | `F11`                  | Переводит выделенный текст, когда включён Translate и нет записи или другого активного действия с выделением. |
| **Prettify выделенный текст**                   | `F12`                  | Улучшает выделенный текст, когда включён Prettify и нет записи или другого активного действия с выделением.   |
| **Повторить расшифровку**                       | `Ctrl+F8`              | Повторяет последнюю расшифровку, доступную для повтора, только когда GPT-Voice бездействует.                  |

Сочетание для повтора недоступно, пока нет расшифровки, доступной для повтора. Смотрите [запись и
расшифровку](../guides/transcription.md), чтобы узнать, когда доступен повтор, и [перевод и Prettify выделенного
текста](../guides/text-actions.md) для действий с выделенным текстом.

## Изменение сочетания клавиш

1. Выберите **Change** в строке действия.
2. В диалоге захвата нажмите полную комбинацию клавиш, которую хотите использовать.
3. Проверьте комбинацию, показанную в диалоге, затем выберите **Apply**. Выберите **Cancel**, чтобы оставить текущее
   сочетание без изменений.

GPT-Voice временно приостанавливает все глобальные сочетания, пока открыт диалог захвата, затем снова регистрирует их
после применения или отмены. Нажмите немодифицирующую клавишу как часть комбинации; нажатие только `Ctrl`, `Alt`,
`Shift` или клавиши Command платформы не создаёт сочетание.

## Как избежать конфликтов

Используйте отдельное сочетание для каждого действия GPT-Voice и выбирайте комбинации, которые не конфликтуют с вашей
операционной системой или другим программным обеспечением. GPT-Voice отклоняет конфликтующие назначения внутри
GPT-Voice. Клавиша без модификатора конфликтует с той же основной клавишей, даже если другое назначение содержит
модификаторы, поэтому не назначайте, например, F9 вместе с Ctrl+F9.

Если новое назначение не удаётся зарегистрировать, GPT-Voice сохраняет текущее сочетание и показывает причину. В
macOS захваченная клавиша Command отображается как `Command`; на других поддерживаемых платформах используется
эквивалент платформы.

## Включение и отключение действий с выделенным текстом

У строк **Translate** и **Prettify** есть переключатель включения. Оба действия включены по умолчанию. Выключите
переключатель, чтобы действие не выполнялось даже при нажатии его сочетания; включите его, чтобы настроенное
сочетание снова стало доступно, когда выполнены обычные условия действия.

Эти переключатели входят в форму Settings, поэтому после их изменения выберите **Save changes**. Изменение сочетания
через диалог захвата применяется независимо; используйте индикатор несохранённых изменений из обзора Settings, чтобы
отличить ожидающие изменения формы от уже применённого сочетания.
