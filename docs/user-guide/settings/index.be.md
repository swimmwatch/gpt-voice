# Агляд налад

Адкрыйце **Settings** з меню трэя GPT-Voice. Акно Settings адкрываецца на **Shortcuts** і мае чатыры раздзелы:

| Раздзел                     | Для чаго выкарыстоўваць                                                                                    |
| --------------------------- | ---------------------------------------------------------------------------------------------------------- |
| **Shortcuts**               | Абраць глабальныя спалучэнні для запісу, спынення, адмены, перакладу, Prettify і паўтору расшыфроўкі.      |
| **[Prettify](prettify.md)** | Абраць пастаўшчыка апрацоўкі тэксту Ollama або vLLM, наладзіць яго мадэль і паводзіны генерацыі.           |
| **[Browser](browser.md)**   | Наладзіць асобу браўзера і фонавыя паводзіны, якія GPT-Voice выкарыстоўвае для службаў на аснове браўзера. |
| **[Network](network.md)**   | Наладзіць проксі, які выкарыстоўваюць гэтыя службы на аснове браўзера.                                     |

Кнопкі раздзелаў застаюцца даступнымі ў вузкіх вокнах як значкі з даступнымі меткамі. Абярыце раздзел, каб змяніць
яго налады; кожная старонка апісвае ўласныя палі і перадумовы.

## Захаванне змяненняў

Калі акно адкрываецца, налады загружаюцца з захаванай канфігурацыі праграмы. Змена значэння пазначае форму станам
**Unsaved changes**. Кнопка **Save changes** даступная толькі пасля змены хаця б аднаго значэння і тады, калі ўсе
бягучыя значэнні палёў сапраўдныя. Пакуль GPT-Voice захоўвае даныя, кнопка застаецца адключанай.

Калі значэнне не адпавядае патрабаванням, адпаведнае поле паказвае паведамленне праверкі, і захаванне блакіруецца,
пакуль вы не выправіце значэнне. Калі памылкай завяршаецца сама аперацыя захавання, Settings застаецца адкрытым і
паказвае паведамленне пра памылку, каб вы маглі выправіць праблему або паўтарыць спробу. Паспяховае захаванне
абнаўляе захаваную канфігурацыю і закрывае акно Settings.

## Закрыйце акно без выпадковай страты працы

Settings без незахаваных змяненняў закрываецца адразу. Калі змяненні ёсць, GPT-Voice пытаецца, ці хочаце вы іх
адкінуць. Абярыце **Keep editing**, каб вярнуцца да формы, або **Discard changes**, каб закрыць Settings без
захавання чаканых змяненняў. Пакуль ідзе захаванне, закрыццё Settings блакіруецца да завяршэння аперацыі.

Гэтае пацвярджэнне тычыцца формы налад. Захоп новага глабальнага спалучэння клавіш — асобнае дзеянне: GPT-Voice
часова прыпыняе глабальныя спалучэнні, пакуль слухае камбінацыю клавіш, а потым аднаўляе іх, калі захоп завершаны
або адменены.
