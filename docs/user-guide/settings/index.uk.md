# Огляд налаштувань

Відкрийте **Settings** з меню трея GPT-Voice. Вікно Settings відкривається на **Shortcuts** і має чотири розділи:

| Розділ                      | Використовуйте для                                                                                                |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **Shortcuts**               | Вибору глобальних сполучень для записування, зупинення, скасування, перекладу, Prettify та повтору розшифрування. |
| **[Prettify](prettify.md)** | Вибору постачальника обробки тексту Ollama або vLLM і налаштування моделі та поведінки генерування.               |
| **[Browser](browser.md)**   | Налаштування ідентичності й фонової поведінки браузера, які GPT-Voice використовує для браузерних сервісів.       |
| **[Network](network.md)**   | Налаштування проксі, який використовують ці браузерні сервіси.                                                    |

Кнопки розділів залишаються доступними у вузьких вікнах як іконки з доступними підписами. Виберіть розділ, щоб змінити
його налаштування; кожна сторінка описує власні поля та передумови.

## Збереження змін

Налаштування завантажуються зі збереженої конфігурації програми, коли відкривається вікно. Зміна значення позначає
форму як таку, що має **Unsaved changes**. Кнопка **Save changes** доступна лише після внесення принаймні однієї зміни
та коли всі поточні значення полів дійсні. Поки GPT-Voice зберігає дані, кнопка лишається вимкненою.

Коли значення не відповідає вимогам, відповідне поле показує повідомлення перевірки, а збереження блокується, доки ви
не виправите його. Якщо не вдається сама операція збереження, Settings залишається відкритим і показує повідомлення
про помилку, щоб ви могли виправити проблему або повторити спробу. Успішне збереження оновлює збережену конфігурацію
та закриває вікно Settings.

## Закрийте вікно, не втративши роботу випадково

Закриття Settings без незбережених змін закриває його відразу. Якщо є незбережені зміни, GPT-Voice запитує, чи хочете
ви їх відкинути. Виберіть **Keep editing**, щоб повернутися до форми, або **Discard changes**, щоб закрити Settings
без збереження змін, що очікують. Поки триває збереження, закриття Settings блокується до завершення операції.

Це підтвердження стосується форми налаштувань. Захоплення нового глобального сполучення клавіш — окрема дія:
GPT-Voice тимчасово призупиняє глобальні сполучення, поки прослуховує комбінацію клавіш, а потім відновлює їх, коли
захоплення завершено або скасовано.
