# Налады пастаўшчыка

Абярыце пастаўшчыка расшыфроўкі ў Command Dock GPT-Voice. Каб наладзіць бягучага выбранага пастаўшчыка, адкрыйце
яго элемент **Connect** або **Configure** у загалоўку. Дыялог пастаўшчыка паказвае толькі элементы кіравання,
прыдатныя для гэтага пастаўшчыка.

## ChatGPT Web

ChatGPT Web выкарыстоўвае сеанс браўзера, а не ключ API. Яго дыялог пастаўшчыка паказвае, ці захаваны сеанс.

1. Абярыце **ChatGPT Web** у Command Dock.
2. Адкрыйце яго дыялог пастаўшчыка і абярыце **Log in**, калі сеанс не захаваны, або **Log in again**, каб замяніць
   захаваны сеанс.
3. Завяршыце ўваход у акне браўзера GPT-Voice, а потым вярніцеся ў Command Dock, калі пастаўшчык пакажа
   **Connected**.

Выкарыстоўвайце **Clear session**, калі хочаце, каб GPT-Voice забыў захаваную аўтэнтыфікацыю ChatGPT Web. Перад
ачысткай GPT-Voice просіць пацвярджэнне. Ачыстка сеанса адключае гэтага пастаўшчыка; увайдзіце зноў перад наступнай
расшыфроўкай.

Звесткі пра ўладальніка ўліковага запісу, сеанс браўзера і абмежаванні пастаўшчыка глядзіце ў
[раздзеле пра пастаўшчыкаў](../guides/providers.md).

## OpenAI API

Абярыце **OpenAI API** і адкрыйце яго дыялог пастаўшчыка, каб наладзіць гэтыя палі. Выкарыстоўвайце свой уласны
ключ API і ўліковы запіс OpenAI; GPT-Voice не дае ключ, крэдыты або доступ да службаў OpenAI.

| Поле            | Бягучыя паводзіны                                                                                                                          |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| **API key**     | Увядзіце новы ключ, каб захаваць яго. Пры паўторным адкрыцці поле пустое, нават калі ключ захаваны; пустое захаванне не замяняе гэты ключ. |
| **Model**       | `whisper-1` — адзіная даступная мадэль расшыфроўкі. Яна паказваецца ў полі толькі для чытання.                                             |
| **Language**    | Абярыце аўтаматычнае вызначэнне (па змаўчанні), англійскую, рускую, украінскую або беларускую мову.                                        |
| **Prompt**      | Неабавязковая падказка для расшыфроўкі. Па змаўчанні поле пустое; прабелы ў пачатку і канцы выдаляюцца пры захаванні.                      |
| **Temperature** | Кіруе варыятыўнасцю расшыфроўкі ад `0` да `1`. Па змаўчанні — `0`; элемент змяняе значэнне крокам `0.05`.                                  |

Абярыце **Save**, каб праверыць і захаваць змяненні. Паспяховае захаванне закрывае дыялог. Недапушчальныя
значэнні мадэлі, мовы або тэмпературы адхіляюцца; калі захаванне не ўдаецца, дыялог паказвае бяспечнае паведамленне
пра памылку і застаецца адкрытым.

## Захаваныя ўліковыя даныя і ачыстка аўтэнтыфікацыі

GPT-Voice захоўвае ключ OpenAI API толькі праз Electron safe storage. Сам ключ не паказваецца зноў у дыялогу;
замест гэтага дыялог паведамляе, што ключ API захаваны. Калі бяспечнае сховішча недаступнае, GPT-Voice не можа
захаваць новы ключ.

Выкарыстоўвайце **Clear API key** і пацвердзіце дыялог, каб выдаліць захаваны ключ, пакінуўшы іншыя налады OpenAI
API. Кнопка даступная, толькі калі ключ захаваны. Перад новай расшыфроўкай праз OpenAI API трэба ўвесці і захаваць
новы ключ.

Уліковыя даныя пастаўшчыка і выкарыстанне службы падпарадкоўваюцца ўмовам, аплаце, квотам і палітыцы
канфідэнцыяльнасці ўліковага запісу пастаўшчыка. Не ўстаўляйце ключы ў запыты ў падтрымку, здымкі экрана або
дакументацыю.
