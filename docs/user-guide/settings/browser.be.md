# Налады браўзера

GPT-Voice выкарыстоўвае CloakBrowser для браўзерных службаў, такіх як ChatGPT Web і пераклад вылучанага тэксту.
Адкрыйце **Settings** і абярыце **Browser**, каб задаць паводзіны і значэнні асобы, якія GPT-Voice выкарыстоўвае пры
стварэнні кантэкстаў браўзера. Проксі наладжваецца ў [наладах Network](network.md).

## Паводзіны браўзера

| Налада                 | Па змаўчанні | Даступныя значэнні          | Вынік                                                           |
| ---------------------- | ------------ | --------------------------- | --------------------------------------------------------------- |
| **Humanize input**     | Уключана     | Уключана або выключана      | Перадае ў CloakBrowser наладу імітацыі ўводу.                   |
| **Human preset**       | **Careful**  | **Default** або **Careful** | Абірае перадустаноўку імітацыі чалавечых паводзін CloakBrowser. |
| **Background browser** | **Hidden**   | **Hidden** або **Visible**  | Вызначае, ці схаваны пастаянны фонавы браўзер GPT-Voice.        |

**Background browser** прымяняецца да пастаяннага фонавага браўзера. Акно ўваходу ChatGPT Web заўсёды бачна, каб вы
маглі завяршыць аўтэнтыфікацыю. Абярыце **Visible**, калі трэба назіраць за фонавым браўзерам; інакш пакіньце
**Hidden**.

## Асоба браўзера

Адкрыйце **Identity**, каб праглядзець або змяніць значэнні, якія GPT-Voice перадае ў кантэкст браўзера.

| Налада               | Па змаўчанні                                      | Патрабаванне і дзеянне                                                                                                                                                      |
| -------------------- | ------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Fingerprint seed** | Лічбавы seed, створаны GPT-Voice                  | Толькі лічбы. Абярыце **Reset**, каб стварыць новы пяцізначны лічбавы seed.                                                                                                 |
| **Locale**           | `en-US`                                           | Абярыце лакаль браўзера: `en-US`, `en-GB`, `ru-RU`, `uk-UA`, `be-BY`, `de-DE`, `fr-FR`, `es-ES`, `it-IT`, `pt-BR`, `pl-PL`, `tr-TR`, `ja-JP`, `ko-KR`, `zh-CN` або `zh-TW`. |
| **Timezone**         | Часавы пояс сістэмы або `UTC`, калі ён недаступны | Абярыце падтрыманы часавы пояс IANA.                                                                                                                                        |

Fingerprint seed, Locale і Timezone абавязковыя. Settings адхіляе seed з не-лічбавымі сімваламі, Locale, якая не
з'яўляецца сапраўднай лакаллю BCP 47, або Timezone, якая не з'яўляецца сапраўдным часавым поясам IANA. Пры захаванні
GPT-Voice прыбірае навакольныя прабелы.

### Proxy GeoIP кіруе Locale і Timezone

Калі проксі ўключаны з **GeoIP** у [наладах Network](network.md), проксі вызначае Locale і Timezone браўзера.
GPT-Voice адключае гэтыя два палі ў **Identity** і паказвае **Proxy GeoIP controls locale and timezone**. Захаваныя
Locale і Timezone застаюцца даступнымі пасля выключэння GeoIP, але не перадаюцца ў кантэкст браўзера, пакуль актыўны
GeoIP проксі кіруе імі.

## Захаванне змяненняў

Налады Browser уваходзяць у форму Settings. Змена значэння стварае **Unsaved changes**; абярыце **Save changes**
пасля паспяховай праверкі. Звычайная канфігурацыя Browser захоўваецца ў лакальных наладах GPT-Voice і выкарыстоўваецца
пры наступным стварэнні адпаведнага кантэксту браўзера.
