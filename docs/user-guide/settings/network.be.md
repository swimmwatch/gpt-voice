# Налады сеткі

Адкрыйце **Settings** і абярыце **Network**, каб наладзіць проксі, які GPT-Voice перадае кантэкстам CloakBrowser.
Гэтыя налады ўплываюць на ChatGPT Web і пераклад вылучанага тэксту. Пра асобу браўзера, на якую ўплывае Proxy GeoIP,
глядзіце [налады Browser](browser.md).

## Уключыце проксі

**Proxy enabled** па змаўчанні выключаны. Калі ён выключаны, GPT-Voice не перадае проксі ў CloakBrowser, а астатнія
палі Network адключаныя. Выключэнне не сцірае ўведзеныя значэнні, таму проксі можна ўключыць пазней.

Калі проксі ўключаны, **Proxy server** абавязковы. Увядзіце поўны URL з адным з пратаколаў:

- `http://`
- `https://`
- `socks5://`

Напрыклад, `http://proxy.example.com:8080` — дапушчальны фармат сервера. GPT-Voice прыбірае навакольныя прабелы і
адхіляе адсутны або няправільны сервер, непадтрыманыя пратаколы і URL з імем карыстальніка або паролем. Укажыце
ўліковыя даныя ў асобных палях.

## Выключэнні і ўліковыя даныя

| Поле         | Па змаўчанні       | Паводзіны                                                                   |
| ------------ | ------------------ | --------------------------------------------------------------------------- |
| **Bypass**   | Пуста              | Неабавязкова. Значэнне выключэння перадаецца ў CloakBrowser разам з проксі. |
| **Username** | Пуста              | Неабавязковае імя карыстальніка для HTTP або HTTPS-аўтэнтыфікацыі проксі.   |
| **Password** | Пароль не захаваны | Неабавязковы пароль для HTTP або HTTPS-аўтэнтыфікацыі проксі.               |

Пароль захоўваецца асобна праз Electron safe storage. Пасля захавання яго значэнне не вяртаецца ў Settings; поле
паказвае, што пароль захаваны. Пустое поле захоўвае існы пароль; абярыце **Clear**, каб выдаліць яго. Калі бяспечнае
сховішча недаступнае, GPT-Voice не можа захаваць новы пароль проксі. Не ўказвайце імя карыстальніка або пароль у URL
Proxy server і не раскрывайце ўліковыя даныя ў запыце ў падтрымку або на здымку экрана.

### Уліковыя даныя SOCKS5 не падтрымліваюцца

CloakBrowser не падтрымлівае імя карыстальніка або пароль для проксі SOCKS5. Калі ва ўключанага SOCKS5-проксі ёсць
хоць адны ўліковыя даныя, Settings паказвае папярэджанне і блакіруе захаванне, пакуль не выдаліце імя карыстальніка
і не ачысціце пароль. GPT-Voice не перадае ўліковыя даныя SOCKS5 у CloakBrowser.

## Дазвольце Proxy GeoIP кіраваць асобай браўзера

**GeoIP** па змаўчанні выключаны і даступны толькі калі проксі ўключаны. Уключыце яго, калі наладжаны проксі павінен
вызначаць Locale і Timezone браўзера. Пакуль проксі і GeoIP актыўныя, GPT-Voice перадае проксі з уключаным GeoIP і
не перадае асобна захаваныя Locale або Timezone.

Таму палі **Locale** і **Timezone** у [наладах Browser](browser.md) адключаныя.

Яны паказваюць **Proxy GeoIP controls locale and timezone**. Выключыце GeoIP, каб зноў рэдагаваць і выкарыстоўваць
захаваныя значэнні асобы Browser.

## Захаванне і вырашэнне праблем

Значэнні Network ўваходзяць у форму Settings. Абярыце **Save changes** пасля паспяховай праверкі; захаваная
канфігурацыя проксі выкарыстоўваецца пры наступным стварэнні адпаведнага кантэксту браўзера GPT-Voice. Калі захаванне
не ўдаецца, Settings застаецца адкрытым і пазначае недапушчальнае поле.

Пры памылцы падлучэння праверце, што URL сервера ўключае `http`, `https` або `socks5`, проксі даступны, а ўліковыя
даныя HTTP/HTTPS унесены ў прызначаныя палі. Не ўводзьце ўліковыя даныя SOCKS5.
