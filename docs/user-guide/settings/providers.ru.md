# Настройки поставщика

Выберите поставщика расшифровки на панели команд GPT-Voice. Чтобы настроить выбранного поставщика, откройте его
элемент **Connect** или **Configure** в заголовке. Диалог поставщика показывает только элементы управления, применимые
к этому поставщику.

## ChatGPT Web

ChatGPT Web использует сеанс браузера, а не API-ключ. Его диалог поставщика показывает, сохранён ли сеанс.

1. Выберите **ChatGPT Web** на панели команд.
2. Откройте его диалог поставщика и выберите **Log in**, когда сеанс не сохранён, или **Log in again**, чтобы заменить
   сохранённый сеанс.
3. Завершите вход в окне браузера GPT-Voice, затем вернитесь к панели команд, когда поставщик покажет **Connected**.

Используйте **Clear session**, когда хотите, чтобы GPT-Voice забыл сохранённую аутентификацию ChatGPT Web. Перед
очисткой GPT-Voice запрашивает подтверждение. Очистка сеанса отключает этого поставщика; перед следующей
расшифровкой войдите снова.

Сведения о владении учётной записью, сеансе браузера и ограничениях поставщика приведены в разделе
[поставщики](../guides/providers.md).

## OpenAI API

Выберите **OpenAI API** и откройте его диалог поставщика, чтобы настроить эти поля. Используйте собственные API-ключ
и учётную запись OpenAI; GPT-Voice не предоставляет ключ, кредиты или доступ к сервисам OpenAI.

| Поле            | Текущее поведение                                                                                                                           |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| **API key**     | Введите новый ключ, чтобы сохранить его. При повторном открытии поле пусто даже при сохранённом ключе, а пустое сохранение не заменяет его. |
| **Model**       | `whisper-1` — единственная доступная модель расшифровки. Она показана в поле только для чтения.                                             |
| **Language**    | Выберите автоматическое определение (по умолчанию), английский, русский, украинский или белорусский.                                        |
| **Prompt**      | Необязательная подсказка для расшифровки. По умолчанию поле пусто; пробелы в начале и конце удаляются при сохранении.                       |
| **Temperature** | Управляет вариативностью расшифровки от `0` до `1`. По умолчанию — `0`; элемент меняет значение с шагом `0.05`.                             |

Выберите **Save**, чтобы проверить и сохранить изменения. Успешное сохранение закрывает диалог. Недопустимые значения
модели, языка или температуры отклоняются; если сохранение завершается ошибкой, диалог показывает безопасное сообщение
об ошибке и остаётся открытым.

## Сохранённые учётные данные и очистка аутентификации

GPT-Voice хранит API-ключ OpenAI только через Electron safe storage. Сам ключ не показывается обратно в диалоге;
вместо этого диалог сообщает, что API-ключ сохранён. Если защищённое хранилище недоступно, GPT-Voice не может
сохранить новый ключ.

Используйте **Clear API key** и подтвердите диалог, чтобы удалить сохранённый ключ, сохранив остальные настройки
OpenAI API. Кнопка доступна, только когда ключ сохранён. Перед следующей расшифровкой через OpenAI API необходимо
ввести и сохранить новый ключ.

Учётные данные поставщика и использование сервиса остаются в рамках условий, оплаты, квот и политики
конфиденциальности учётной записи поставщика. Не вставляйте ключи в запросы в поддержку, снимки экрана или
документацию.
