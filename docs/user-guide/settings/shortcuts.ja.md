# ショートカット設定

GPT-Voice はグローバル ショートカットを使用するため、別のアプリケーションを使用しているときにも機能できます。 **Settings** を開き、**Shortcuts** を選択して表示または変更します。

| アクション                            | デフォルトのショートカット | うまくいくとき                                                                                                       |
| ------------------------------------- | -------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| **Start, pause, or resume recording** | `F9`                       | アイドル時に録画を開始したり、アクティブな録画を一時停止したり、一時停止した録画を再開したりします。                 |
| **Stop recording**                    | `F10`                      | 録音または一時停止した録音を停止し、文字起こしを開始します。                                                         |
| **Cancel**                            | `Escape`                   | アクティブな録音をキャンセルします。アクティブな録音がない場合は、実行中の Prettify リクエストがキャンセルされます。 |
| **Translate selected text**           | `F11`                      | 翻訳が有効で、録音や他の選択テキスト アクションがアクティブでない場合、選択したテキストを翻訳します。                |
| **Prettify selected text**            | `F12`                      | Prettify が有効で、録音や他の選択テキスト アクションがアクティブでない場合、選択したテキストをきれいにします。       |
| **Retry transcription**               | `Ctrl+F8`                  | GPT-Voice がアイドル状態の場合にのみ、最新の再試行可能な転写を再試行します。                                         |

再試行可能な文字起こしができるまで、再試行ショートカットは使用できません。再試行が可能になる場合については [録音と転写](../guides/transcription.md)] を、選択したテキストのワークフローについては [選択したテキストの翻訳と整形 ](../guides/text-actions.md) を参照してください。

## ショートカットを変更する

1. アクションの行で **Change** を選択します。
2. キャプチャダイアログで、使用する完全なキーの組み合わせを押します。
3. ダイアログに表示される組み合わせを確認し、**Apply**を選択します。現在のショートカットを変更しない場合は、**Cancel** を選択します。

GPT-Voice は、キャプチャ ダイアログが開いている間、すべてのグローバル ショートカットを一時的に停止し、適用またはキャンセルするとそれらを再度登録します。組み合わせの一部として非修飾キーを押します。 `Ctrl`、`Alt`、`Shift`、またはプラットフォームのコマンド キーのみを押しても、ショートカットは作成されません。

## 競合を避ける

GPT-Voice アクションごとに異なるショートカットを使用し、オペレーティング システムや他のソフトウェアと衝突しない組み合わせを選択してください。 GPT-Voice は、競合する GPT-Voice の割り当てを拒否します。変更されていないキーは、他の割り当てに修飾子が含まれている場合でも、同じベース キーと競合するため、たとえば、F9 と Ctrl+F9 を組み合わせないでください。

新しい割り当てを登録できない場合、GPT-Voice は現在のショートカットを保持し、その理由を表示します。 macOS では、キャプチャされた Command キーは `Command` として表されます。他のサポートされているプラ​​ットフォームでは、同等のプラットフォームが使用されます。## 選択したテキストのアクションを有効または無効にする

**Translate** 行と **Prettify** 行にはそれぞれイネーブル スイッチがあります。どちらもデフォルトで有効になっています。スイッチをオフにすると、ショートカットが押されてもそのアクションが実行されなくなります。これをオンにすると、アクションの通常の条件が満たされたときに、設定されたショートカットが再び使用可能になります。

これらのスイッチは設定フォームの一部であるため、変更した後 **Save changes** を選択してください。キャプチャ ダイアログを通じてショートカットを変更すると、そのショートカットが個別に適用されます。設定概要の未保存変更インジケーターを使用して、保留中のフォーム編集と既に適用されているショートカットを区別します。
