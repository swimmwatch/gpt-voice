# Настройки браузера

GPT-Voice использует CloakBrowser для браузерных служб, таких как ChatGPT Web и перевод выделенного текста. Откройте
**Settings** и выберите **Browser**, чтобы задать значения поведения и идентичности, которые GPT-Voice использует при
создании этих контекстов браузера. Прокси настраивается в [настройках Network](network.md).

## Поведение браузера

| Настройка              | По умолчанию | Доступные значения          | Эффект                                                                    |
| ---------------------- | ------------ | --------------------------- | ------------------------------------------------------------------------- |
| **Humanize input**     | Включено     | Включено или выключено      | Передаёт CloakBrowser настройку имитации ввода.                           |
| **Human preset**       | **Careful**  | **Default** или **Careful** | Выбирает предустановку имитации человеческого поведения CloakBrowser.     |
| **Background browser** | **Hidden**   | **Hidden** или **Visible**  | Управляет тем, скрыт ли постоянный фоновый браузер GPT-Voice или показан. |

Настройка **Background browser** применяется к постоянному фоновому браузеру. Окно входа ChatGPT Web всегда видно,
чтобы вы могли завершить аутентификацию. Выберите **Visible**, когда нужно наблюдать за фоновым браузером; в ином
случае оставьте режим **Hidden** по умолчанию.

## Идентичность браузера

Откройте **Identity**, чтобы просмотреть или изменить значения, передаваемые GPT-Voice в контекст браузера.

| Настройка            | По умолчанию                                           | Требование и действие                                                                                                                                                                                |
| -------------------- | ------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Fingerprint seed** | Числовое начальное значение, сгенерированное GPT-Voice | Обязательны только цифры. Выберите **Reset**, чтобы сгенерировать новое пятизначное числовое начальное значение.                                                                                     |
| **Locale**           | `en-US`                                                | Выберите один из поддерживаемых локалей браузера: `en-US`, `en-GB`, `ru-RU`, `uk-UA`, `be-BY`, `de-DE`, `fr-FR`, `es-ES`, `it-IT`, `pt-BR`, `pl-PL`, `tr-TR`, `ja-JP`, `ko-KR`, `zh-CN` или `zh-TW`. |
| **Timezone**         | Часовой пояс системы или `UTC`, если он недоступен     | Выберите поддерживаемый часовой пояс IANA.                                                                                                                                                           |

Fingerprint seed, Locale и Timezone обязательны. Settings отклоняет seed, содержащий что-либо кроме цифр, Locale,
который не является допустимой локалью BCP 47, или Timezone, который не является допустимым часовым поясом IANA.
При сохранении GPT-Voice удаляет окружающие пробелы из этих значений.

### Proxy GeoIP управляет Locale и Timezone

Когда прокси включён с **GeoIP** в [настройках Network](network.md), прокси определяет Locale и Timezone браузера.
GPT-Voice отключает эти два поля в **Identity** и показывает сообщение **Proxy GeoIP controls locale and timezone**.
Сохранённые Locale и Timezone остаются доступны, если позже выключить GeoIP, но не передаются в контекст браузера,
пока активный GeoIP прокси управляет ими.

## Сохранение изменений

Настройки Browser входят в форму Settings. Изменение значения создаёт **Unsaved changes**; выберите **Save changes**
после успешной проверки. Обычная конфигурация Browser хранится в локальных настройках GPT-Voice и используется при
следующем создании применимого контекста браузера. О сохранении с ошибкой и подтверждении отмены смотрите
[обзор настроек](index.md).
