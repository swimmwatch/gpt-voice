# Настройки сети

Откройте **Settings** и выберите **Network**, чтобы настроить прокси, который GPT-Voice передаёт контекстам
CloakBrowser. Эти настройки влияют на браузерные службы, такие как ChatGPT Web и перевод выделенного текста. О
настройках идентичности, на которые влияет Proxy GeoIP, смотрите [настройки Browser](browser.md).

## Включение прокси

**Proxy enabled** по умолчанию выключен. Когда он выключен, GPT-Voice не передаёт прокси в CloakBrowser, а остальные
поля Network отключены. Выключение не удаляет введённые значения, поэтому прокси можно включить снова позднее.

Когда прокси включён, **Proxy server** обязателен. Введите полный URL, использующий один из этих протоколов:

- `http://`
- `https://`
- `socks5://`

Например, `http://proxy.example.com:8080` — допустимый формат сервера. При сохранении GPT-Voice удаляет окружающие
пробелы. Он отклоняет отсутствующий или неправильно сформированный сервер, неподдерживаемые протоколы и URL с именем
пользователя или паролем. Укажите учётные данные в отдельных полях.

## Исключения и учётные данные

| Поле         | По умолчанию       | Поведение                                                                                    |
| ------------ | ------------------ | -------------------------------------------------------------------------------------------- |
| **Bypass**   | Пусто              | Необязательно. Если указано, GPT-Voice передаёт значение исключения в CloakBrowser с прокси. |
| **Username** | Пусто              | Необязательное имя пользователя прокси для аутентификации HTTP или HTTPS.                    |
| **Password** | Пароль не сохранён | Необязательный пароль прокси для аутентификации HTTP или HTTPS.                              |

Пароль хранится отдельно через Electron safe storage. После сохранения его значение не возвращается в Settings;
вместо этого поле показывает, что пароль сохранён. Пустое поле сохраняет существующий пароль. Выберите **Clear**,
чтобы удалить его. Если защищённое хранилище недоступно, GPT-Voice не может сохранить новый пароль прокси.

Не указывайте имя пользователя или пароль в URL Proxy server, не вставляйте учётные данные в запрос в поддержку и не
показывайте их на снимке экрана.

### Учётные данные SOCKS5 не поддерживаются

CloakBrowser не поддерживает имя пользователя или пароль для прокси SOCKS5. Когда у включённого прокси SOCKS5 есть
хотя бы одно учётное данное, Settings показывает предупреждение и блокирует сохранение, пока не удалите имя
пользователя и не очистите пароль. GPT-Voice не передаёт учётные данные SOCKS5 в CloakBrowser.

## Передача идентичности браузера Proxy GeoIP

**GeoIP** по умолчанию выключен и доступен, только когда включён прокси. Включите его, когда настроенный прокси должен
определять Locale и Timezone браузера. Пока прокси и GeoIP активны, GPT-Voice передаёт прокси с включённым GeoIP и не
передаёт отдельно сохранённые Locale или Timezone.

Следовательно, поля **Locale** и **Timezone** в [настройках Browser](browser.md) отключены. В них показано сообщение:

**Proxy GeoIP controls locale and timezone**

Выключите GeoIP, чтобы снова редактировать и использовать сохранённые значения идентичности Browser.

## Сохранение и устранение неполадок

Значения Network входят в форму Settings. Выберите **Save changes** после успешной проверки; сохранённая конфигурация
прокси используется при следующем создании применимого контекста браузера GPT-Voice. Если сохранение завершается
ошибкой, Settings остаётся открытым и указывает недопустимое поле.

При ошибке подключения убедитесь, что URL сервера включает `http`, `https` или `socks5`, прокси доступен, а учётные
данные HTTP/HTTPS указаны в предназначенных для них полях. Не вводите учётные данные SOCKS5. О несохранённых
изменениях и подтверждении отмены смотрите [обзор настроек](index.md).
