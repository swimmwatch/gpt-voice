# 设置概述

从 GPT-Voice 托盘菜单中打开 **Settings**。设置窗口在 **Shortcuts** 上打开，有四个部分：

| 部分                        | 用它来                                                       |
| --------------------------- | ------------------------------------------------------------ |
| **Shortcuts**               | 选择用于录制、停止、取消、翻译、美化和重试转录的全局快捷键。 |
| **[Prettify](prettify.md)** | 选择 Ollama 或 vLLM 文本处理提供程序并配置其模型和生成行为。 |
| **[Browser](browser.md)**   | 配置用于其基于浏览器的服务的浏览器身份和后台行为 GPT-Voice。 |
| **[Network](network.md)**   | 配置这些基于浏览器的服务使用的代理。                         |

部分按钮在狭窄的窗口上仍然可用，作为带有可访问标签的图标。选择一个部分来更改其设置；每个页面都描述了自己的领域和先决条件。

## 保存更改

窗口打开时，将从保存的应用程序配置中加载设置。更改值会将表单标记为具有 **Unsaved changes**。仅当至少进行一项更改且所有当前字段值均有效后，**Save changes** 按钮才可用。当 GPT-Voice 保存时，它保持禁用状态。

当值不满足其要求时，受影响的字段会显示一条验证消息，并且保存将被阻止，直到您更正它。如果保存操作本身失败，“设置”将保持打开状态并显示错误消息，以便您可以更正问题或重试。成功保存将更新保存的配置并关闭“设置”窗口。

## 关闭而不意外丢失工作

关闭没有未保存更改的设置会立即将其关闭。如果有未保存的更改，GPT-Voice 会询问您是否要放弃它们。选择 **Keep editing** 返回表单，或选择 **Discard changes** 关闭设置而不保存挂起的编辑。当保存正在进行时，关闭“设置”将被阻止，直到操作结束。

此确认适用于设置表单。捕获新的全局快捷方式是一个单独的操作：GPT-Voice 在侦听组合键时暂时暂停全局快捷方式，然后在捕获完成或取消时恢复它们。
