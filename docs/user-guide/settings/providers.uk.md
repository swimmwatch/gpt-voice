# Налаштування провайдера

Виберіть постачальника транскрипції в GPT-Voice Command Dock. Щоб налаштувати поточний вибраний постачальник, відкрийте його елемент керування **Connect** або **Configure** у заголовку. У діалоговому вікні постачальника відображаються лише елементи керування, які застосовуються до цього постачальника.

## ChatGPT Web

ChatGPT Web використовує сеанс браузера, а не ключ API. Його діалогове вікно постачальника відображає, чи збережено сеанс.

1. Виберіть **ChatGPT Web** у Command Dock.
2. Відкрийте діалогове вікно постачальника та виберіть **Log in**, якщо сеанс не збережено, або **Log in again**, щоб замінити збережений сеанс.
3. Завершіть вхід у вікні браузера GPT-Voice, а потім поверніться до Command Dock, коли постачальник покаже **Connected**.

Використовуйте **Clear session**, якщо ви хочете GPT-Voice забути збережену ChatGPT Web автентифікацію. GPT-Voice запитує підтвердження перед видаленням. Очищення сеансу відключає цього провайдера; увійдіть знову, перш ніж використовувати його для транскрипції.

Щоб отримати відомості про право власності на обліковий запис, сеанс веб-переглядача та ліміт постачальника, див. [providers](../guides/providers.md).

## OpenAI API

Виберіть **OpenAI API** і відкрийте діалогове вікно постачальника, щоб налаштувати ці поля. Використовуйте власний OpenAI API ключ і обліковий запис; GPT-Voice не надає ключ, кредити або доступ до OpenAI послуг.

| Поле            | Поточна поведінка                                                                                                                                                             |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **API key**     | Введіть новий ключ, щоб зберегти його. Поле залишається порожнім після повторного відкриття, навіть якщо ключ збережено, і порожнє збереження не замінює цей збережений ключ. |
| **Model**       | `whisper-1` — єдина доступна модель транскрипції. Воно відображається як поле лише для читання.                                                                               |
| **Language**    | Виберіть автоматичне визначення (за замовчуванням), англійську, російську, українську або білоруську.                                                                         |
| **Prompt**      | Додаткові вказівки щодо транскрипції. За замовчуванням порожній; пробіли на початку та в кінці видаляються під час збереження.                                                |
| **Temperature** | Керує варіацією транскрипції від 0 до 1. За замовчуванням 0; керування змінюється з кроком 0,05.                                                                              |

Виберіть **Save**, щоб перевірити та зберегти зміни. Успішне збереження закриває діалогове вікно. Недійсні значення моделі, мови або температури відхиляються; якщо зберегти не вдалося, діалогове вікно показує безпечне повідомлення про помилку та залишається відкритим.

## Збережені облікові дані та очищення автентифікації

GPT-Voice зберігає ключ OpenAI API лише через Electron safe storage. Сам ключ не відображається в діалоговому вікні; замість цього діалогове вікно вказує, що ключ API збережено. Якщо безпечне сховище недоступне, GPT-Voice не зможе зберегти новий ключ.Використовуйте **Clear API key** і підтвердьте діалогове вікно, щоб видалити збережений ключ, зберігаючи інші налаштування OpenAI API. Кнопка доступна, лише якщо ключ збережено. Ви повинні ввести та зберегти новий ключ, перш ніж OpenAI API зможе знову транскрибувати.

Облікові дані постачальника та використання послуг залишаються відповідно до умов облікового запису постачальника, виставлення рахунків, квот і політики конфіденційності. Не вставляйте ключі в запити на підтримку, знімки екрана чи документацію.
