# 提供商设置

在 GPT-Voice Command Dock 中选择转录提供程序。要配置当前选定的提供程序，请在标题中打开其 **Connect** 或 **Configure** 控件。提供程序对话框仅显示适用于该提供程序的控件。

## ChatGPT Web

ChatGPT Web 使用浏览器会话而不是 API 密钥。其提供程序对话框显示是否保存会话。

1. 在Command Dock中选择**ChatGPT Web**。
2. 打开其提供程序对话框，如果没有保存会话，请选择 **Log in**，或者选择 **Log in again** 以替换已保存的会话。
3. 在GPT-Voice浏览器窗口中完成登录，然后当提供商显示**Connected**时返回到Command Dock。

当您想要 GPT-Voice 忘记保存的 ChatGPT Web 身份验证时，请使用 **Clear session**。 GPT-Voice 在清除之前要求确认。清除会话会断开此提供者的连接；在使用它进行转录之前再次登录。

有关帐户所有权、浏览器会话和提供商限制的详细信息，请参阅 [providers](../guides/providers.md)。

## OpenAI API

选择 **OpenAI API** 并打开其提供程序对话框来配置这些字段。使用您自己的OpenAI API密钥和帐户； GPT-Voice 不提供密钥、积分或对 OpenAI 服务的访问权限。

| 领域            | 当前行为                                                                                         |
| --------------- | ------------------------------------------------------------------------------------------------ |
| **API key**     | 输入新密钥以保存它。即使存储了密钥，重新打开时该字段仍为空白，并且空白保存不会替换该存储的密钥。 |
| **Model**       | `whisper-1` 是唯一可用的转录模型。它显示为只读字段。                                             |
| **Language**    | 选择自动检测（默认）、英语、俄语、乌克兰语或白俄罗斯语。                                         |
| **Prompt**      | 可选的转录指导。默认为空；保存时，前导和尾随空格将被删除。                                       |
| **Temperature** | 控制转录变异从0到1。默认为0；控制以 0.05 步长变化。                                              |

选择 **Save** 验证并存储更改。成功保存将关闭对话框。无效的型号、语言或温度值将被拒绝；如果保存失败，该对话框会显示一条安全错误消息并保持打开状态。

## 存储凭证和清除身份验证

GPT-Voice仅通过Electron safe storage存储OpenAI API键。键本身不会显示在对话框中；相反，该对话框指示已存储 API 密钥。如果安全存储不可用，GPT-Voice 无法保存新密钥。使用 **Clear API key** 并确认对话框以删除存储的密钥，同时保留其他 OpenAI API 设置。仅当存储密钥时该按钮才可用。您必须输入并保存新密钥，然后 OpenAI API 才能再次转录。

提供商凭证和服务使用仍受提供商帐户的条款、计费、配额和隐私政策的约束。请勿将密钥粘贴到支持请求、屏幕截图或文档中。
