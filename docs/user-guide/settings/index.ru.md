# Обзор настроек

Откройте **Settings** из меню трея GPT-Voice. Окно Settings открывается на разделе **Shortcuts** и содержит четыре
раздела:

| Раздел                      | Назначение                                                                                          |
| --------------------------- | --------------------------------------------------------------------------------------------------- |
| **Shortcuts**               | Выбор глобальных сочетаний для записи, остановки, отмены, перевода, Prettify и повтора расшифровки. |
| **[Prettify](prettify.md)** | Выбор поставщика обработки текста Ollama или vLLM, настройка его модели и поведения генерации.      |
| **[Browser](browser.md)**   | Настройка идентичности браузера и фонового поведения для браузерных служб GPT-Voice.                |
| **[Network](network.md)**   | Настройка прокси, который используют эти браузерные службы.                                         |

Кнопки разделов остаются доступны в узких окнах как значки с доступными метками. Выберите раздел, чтобы изменить его
настройки; на каждой странице описаны свои поля и предварительные условия.

## Сохранение изменений

Когда окно открывается, настройки загружаются из сохранённой конфигурации приложения. Изменение значения помечает
форму состоянием **Unsaved changes**. Кнопка **Save changes** доступна, только когда изменено хотя бы одно значение и
все текущие значения полей действительны. Пока GPT-Voice сохраняет данные, кнопка остаётся отключённой.

Если значение не отвечает требованиям, затронутое поле показывает сообщение о проверке, и сохранение блокируется, пока
вы не исправите значение. Если завершается ошибкой сама операция сохранения, Settings остаётся открытым и показывает
сообщение об ошибке, чтобы вы могли исправить проблему или повторить попытку. Успешное сохранение обновляет сохранённую
конфигурацию и закрывает окно Settings.

## Закрытие без случайной потери работы

Settings без несохранённых изменений закрывается сразу. Если изменения есть, GPT-Voice спрашивает, хотите ли вы их
отбросить. Выберите **Keep editing**, чтобы вернуться к форме, или **Discard changes**, чтобы закрыть Settings без
сохранения ожидающих изменений. Пока идёт сохранение, закрытие Settings блокируется до окончания операции.

Это подтверждение относится к форме настроек. Захват нового глобального сочетания клавиш — отдельное действие:
GPT-Voice временно приостанавливает глобальные сочетания, пока слушает комбинацию клавиш, затем снова включает их после
завершения или отмены захвата.
