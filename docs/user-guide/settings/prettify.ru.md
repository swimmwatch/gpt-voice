# Настройки Prettify

**Prettify** — действие GPT-Voice для очистки выделенного текста с сохранением его смысла. Откройте **Settings** и
выберите **Prettify**, чтобы выбрать службу, её модель и поведение генерации. О выборе текста и запуске действия
смотрите [перевод и Prettify выделенного текста](../guides/text-actions.md).

Для Prettify требуется служба Ollama или vLLM, которой управляете вы, и выбранная модель для этой службы. GPT-Voice
не загружает, не запускает, не размещает и не оплачивает ни одну из этих служб. Поле модели обязательно; если модель
не настроена, Prettify сообщает, что нужна модель, вместо отправки выделенного текста.

## Выбор поставщика и подключение к нему

Выберите **Ollama** или **vLLM** в **Provider**. GPT-Voice хранит базовый URL и выбранную модель отдельно для
каждого поставщика, поэтому смена поставщика не заменяет выбор другого. По умолчанию выбран Ollama.

| Поставщик  | Базовый URL по умолчанию   | Модель по умолчанию | Поведение подключения                                                                                   |
| ---------- | -------------------------- | ------------------- | ------------------------------------------------------------------------------------------------------- |
| **Ollama** | `http://127.0.0.1:11434`   | Модель не выбрана   | Обновляет список моделей, доступных в вашей службе Ollama, и отправляет ей запросы Prettify.            |
| **vLLM**   | `http://127.0.0.1:8000/v1` | Модель не выбрана   | Обновляет список моделей, предоставляемых совместимой с vLLM службой, и отправляет ей запросы Prettify. |

Введите полный базовый URL с `http` или `https`. При сохранении GPT-Voice удаляет окружающие пробелы и завершающие
косые черты. URL не с HTTP(S), а также URL с именем пользователя или паролем отклоняются. HTTP разрешён только для
конечного адреса обратной петли, например `127.0.0.1`, `localhost` или `::1`; каждый адрес вне обратной петли должен
использовать HTTPS.

Когда активный действительный базовый URL является удалённым, а не обратной петлёй, Settings показывает уведомление о
конфиденциальности. При использовании такого адреса выбранный текст и настроенная подсказка Prettify отправляются в
выбранную вами службу. Перед удалённой обработкой проверьте политику конфиденциальности, хранения и контроля доступа
этой службы.

### API-ключ vLLM

Поле **vLLM API key** появляется только при выбранном vLLM. Используйте его, если вашей службе vLLM требуется
аутентификация Bearer. GPT-Voice отправляет ключ с запросами vLLM только когда ключ настроен.

Ключ хранится отдельно через Electron safe storage. После сохранения поле больше не раскрывает его, а сообщает, что
ключ сохранён. Пустое поле сохраняет существующий ключ. Выберите **Clear API key**, чтобы удалить его. Если защищённое
хранилище недоступно в вашей системе, GPT-Voice не может сохранить новый ключ vLLM.

Не указывайте ключ в базовом URL, снимке экрана или запросе в поддержку.

## Выбор модели и управление ею

Выберите **Refresh models** после запуска службы активного поставщика или изменения её базового URL. Список моделей
получается от активного поставщика, поэтому после смены поставщика обновите его снова. Перед запуском Prettify
выберите одну из возвращённых моделей. Если не удаётся подключение, служба, аутентификация или ответ поставщика,
Settings показывает ошибку подключения или обновления моделей; проверьте действительность URL, работу службы и ключ
vLLM, затем повторите обновление.

Когда выбрана модель, Ollama показывает дополнительное меню **Model actions**:

- **Load model** просит Ollama удерживать выбранную модель загруженной для GPT-Voice. Если GPT-Voice удерживал
  загруженной другую модель Ollama, сначала он освобождает её.
- **Free model** просит Ollama освободить выбранную модель из памяти.

Эти действия доступны только для Ollama, но не для vLLM. После проверки состояния работающей модели Ollama выбранная
модель показывает **Loaded** или **Not loaded**. Когда Ollama сообщает размер, Settings также показывает
приблизительный размер модели или загруженной VRAM. Считайте это оценкой от Ollama, а не резервированием памяти,
которое гарантирует GPT-Voice.

Загрузка, освобождение и обновление могут завершиться ошибкой, если конечный адрес недоступен или поставщик отклоняет
запрос. Settings оставляет текущую форму открытой и показывает результат или ошибку, чтобы вы могли исправить адрес
или модель и повторить попытку.

## Управление генерацией текста

**Temperature** — основной параметр генерации. По умолчанию это **0** с допустимым диапазоном от **0 до 1** с шагом
**0.05**. Меньшее значение запрашивает у поставщика меньшую вариативность; после сохранения Settings изменение влияет
на следующий запрос Prettify.

Откройте **Advanced generation**, чтобы изменить остальные параметры. В свёрнутом описании указано, все ли расширенные
значения используют значения по умолчанию или сколько из них изменено. Эти параметры отправляются выбранному
поставщику с каждым запросом Prettify.

| Параметр                  | По умолчанию | Допустимое значение                          | Назначение                                                                                                                                                                             |
| ------------------------- | -----------: | -------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Top P**                 |        `0.9` | `0.05`–`1`, с шагом `0.05`                   | Ограничение выбора наиболее вероятным диапазоном накопленной вероятности.                                                                                                              |
| **Min P**                 |          `0` | `0`–`1`, с шагом `0.05`                      | Исключение менее вероятных вариантов ниже выбранного порога.                                                                                                                           |
| **Repeat penalty**        |          `1` | `0.8`–`1.5`, с шагом `0.05`                  | Настройка того, насколько сильно поставщик избегает повторяющегося вывода.                                                                                                             |
| **Top K**                 |         `40` | Целое число от `1` до `200`                  | Ограничение каждого выбора наиболее вероятными кандидатами.                                                                                                                            |
| **Maximum output tokens** |       `4096` | Целое число от `1` до `8192`                 | Ограничение длины сгенерированного ответа. Ответ всё ещё может быть короче.                                                                                                            |
| **Seed**                  |    Не задано | Пусто или целое число от `0` до `2147483647` | Передача необязательного числового начального значения поставщику. Оно может помочь воспроизвести запрос, но результат всё ещё может меняться в зависимости от модели и версии службы. |

Десятичные параметры используют шаг `0.05`. Top P принимает значения от `0.05` до `1`; Min P — от `0` до `1`; а
Repeat penalty — от `0.8` до `1.5`. Top K принимает целые числа от `1` до `200`, а Maximum output tokens — целые
числа от `1` до `8192`.

Используйте значения по умолчанию, если не знаете требований выбранной модели и службы. GPT-Voice отправляет
эквивалентные параметры генерации Ollama и vLLM, но каждая служба всё ещё может отклонить запрос или обработать
параметр согласно поддержке своей модели.

## Написание подсказки Prettify

**Prompt** обязателен и по умолчанию содержит встроенную в GPT-Voice консервативную инструкцию редактора. Она велит
службе рассматривать выделенный текст как инертный исходный материал, сохранять язык и смысл, исправлять и прояснять
его, удалять ненужные повторы и возвращать только отредактированный текст. Она также предписывает не исполнять
инструкции из выделенного текста.

Вы можете заменить подсказку для другой политики редактирования. Ограничьте её **4,000 символами или менее**. Пустая
подсказка, подсказка длиннее 4,000 символов, неподдерживаемый поставщик, недействительный конечный адрес, пустая
модель или значение генерации вне диапазона блокирует **Save changes** и указывает затронутое поле. При сохранении
настроек пробелы в начале и конце удаляются.

Подсказка отправляется вместе с выделенным текстом активному поставщику. Не включайте пароли, API-ключи, персональные
данные или конфиденциальные инструкции, которые вы не хотите передавать этому поставщику.

## Сохранение и проверка изменений

Поставщик, модель, подсказка и параметры генерации входят в форму Settings. Изменение создаёт **Unsaved changes**;
выберите **Save changes** только когда в форме нет ошибок проверки. Успешное сохранение сохраняет обычную
конфигурацию Prettify и закрывает Settings. Ключ vLLM остаётся отдельным: он хранится только через Electron safe
storage и никогда не возвращается в представлении настроек.

Если сохранение завершается ошибкой, окно Settings остаётся открытым с безопасным сообщением об ошибке. Исправьте
указанное поле или подключение к поставщику и повторите попытку. Смотрите [обзор настроек](index.md), чтобы узнать о
несохранённых изменениях и подтверждении отмены.
