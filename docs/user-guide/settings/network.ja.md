# ネットワーク設定

**Settings** を開き、**Network** を選択して、CloakBrowser コンテキストに渡すプロキシ GPT-Voice を構成します。これらの設定は、ChatGPT Web や選択されたテキストの翻訳などのブラウザベースのサービスに影響します。プロキシ GeoIP の影響を受ける ID 設定については、「ブラウザ設定](browser.md)」を参照してください。

## プロキシを有効にする

**Proxy enabled** はデフォルトではオフです。オフの場合、GPT-Voice はプロキシを CloakBrowser に渡さず、残りのネットワーク フィールドは無効になります。これをオフにしても、入力した値は消去されないため、後で再度プロキシを有効にすることができます。

有効にする場合は**Proxy server**が必要です。次のプロトコルのいずれかを使用して完全な URL を入力します。

- `http://`
- `https://`
- `socks5://`

たとえば、`http://proxy.example.com:8080` は有効なサーバー形式です。 GPT-Voice は、値を保存するときに周囲の空白を削除します。サーバーの欠落または不正な形式、サポートされていないプロトコル、ユーザー名またはパスワードを含む URL は拒否されます。代わりに、資格情報を別のフィールドに入力します。

## バイパスと認証情報

| フィールド   | デフォルト                     | 行動                                                                                         |
| ------------ | ------------------------------ | -------------------------------------------------------------------------------------------- |
| **Bypass**   | 空白                           | オプション。指定すると、GPT-Voice はプロキシを使用してバイパス値を CloakBrowser に渡します。 |
| **Username** | 空白                           | HTTP または HTTPS プロキシ認証用のオプションのプロキシ ユーザー名。                          |
| **Password** | パスワードが保存されていません | HTTP または HTTPS プロキシ認証用のオプションのプロキシ パスワード。                          |

パスワードは Electron safe storage を通じて個別に保存されます。保存後、その値は設定に戻されません。このフィールドには、代わりにパスワードが保存されることが示されています。フィールドを空白のままにすると、既存のパスワードが保持されます。 **Clear** を選択して削除します。安全なストレージが利用できない場合、GPT-Voice は新しいプロキシ パスワードを保存できません。

プロキシ サーバーの URL にユーザー名やパスワードを入力したり、サポート リクエストに資格情報を貼り付けたり、スクリーンショットで公開したりしないでください。

### SOCKS5 資格情報はサポートされていません

CloakBrowser は、SOCKS5 プロキシのユーザー名またはパスワードをサポートしていません。有効な SOCKS5 プロキシにいずれかの資格情報がある場合、設定は警告を表示し、ユーザー名を削除してパスワードをクリアするまで保存をブロックします。 GPT-Voice は SOCKS5 資格情報を CloakBrowser に渡しません。

## プロキシ GeoIP 自身のブラウザ ID を許可します

**GeoIP** はデフォルトではオフであり、プロキシが有効な場合にのみ使用できます。構成されたプロキシがブラウザーのロケールとタイムゾーンを決定する必要がある場合は、これをオンにします。プロキシと GeoIP の両方がアクティブである間、GPT-Voice は GeoIP が有効になったプロキシを渡しますが、個別に保存されたロケールやタイムゾーンは渡しません。その結果、[ブラウザ設定](browser.md)の**Locale**および**Timezone**フィールドは無効になり、**Proxy GeoIP controls locale and timezone**が表示されます。保存されたブラウザ ID 値を再度編集して使用するには、GeoIP をオフにします。

## 保存してトラブルシューティングを行う

ネットワーク値は設定フォームの一部です。検証が成功したら **Save changes** を選択します。保存されたプロキシ設定は、次回 GPT-Voice が該当するブラウザ コンテキストを作成するときに使用されます。保存に失敗した場合、設定は開いたままになり、無効なフィールドが識別されます。

接続が失敗した場合は、サーバー URL に `http`、`https`、または `socks5` が含まれていること、プロキシに到達可能であること、および HTTP/HTTPS 資格情報が専用フィールドにあることを確認してください。 SOCKS5 資格情報を入力しないでください。未保存変更および破棄確認の動作については、[設定の概要](index.md)を参照してください。
