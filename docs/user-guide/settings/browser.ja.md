# ブラウザの設定

GPT-Voice は、ChatGPT Web や選択されたテキストの翻訳などのブラウザベースのサービスに CloakBrowser を使用します。 **Settings** を開き、**Browser** を選択して、GPT-Voice がこれらのブラウザー コンテキストを作成するときに使用される動作と ID 値を設定します。 [ネットワーク設定](network.md)]でプロキシを設定します。

## ブラウザの動作

| 設定                   | デフォルト  | 利用可能な値                   | 効果                                                                                        |
| ---------------------- | ----------- | ------------------------------ | ------------------------------------------------------------------------------------------- |
| **Humanize input**     | 有効        | 有効または無効                 | Humanize 入力設定を CloakBrowser に渡します。                                               |
| **Human preset**       | **Careful** | **Default** または **Careful** | CloakBrowser のヒューマナイゼーション プリセットを選択します。                              |
| **Background browser** | **Hidden**  | **Hidden** または **Visible**  | GPT-Voice の永続的なバックグラウンド ブラウザーをヘッドレスにするか表示するかを制御します。 |

**Background browser** 設定は、永続的なバックグラウンド ブラウザーに適用されます。 ChatGPT Web サインイン ウィンドウが常に表示されるので、認証を完了できます。バックグラウンドのブラウザを観察する必要がある場合は、**Visible** を選択してください。それ以外の場合は、デフォルトの **Hidden** モードが選択されたままになります。

## ブラウザの ID

**Identity** を開いて、GPT-Voice がブラウザー コンテキストに渡す値を表示または変更します。

| 設定                 | デフォルト                                             | 要件とアクション                                                                                                                                                                                                   |
| -------------------- | ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Fingerprint seed** | GPT-Voice が生成した数値シード                         | 必須の数字のみ。新しい 5 桁の数値シードを生成するには、**Reset** を選択します。                                                                                                                                    |
| **Locale**           | `en-US`                                                | サポートされているブラウザ ロケールのいずれかを選択します: `en-US`、`en-GB`、`ru-RU`、`uk-UA`、`be-BY`、`de-DE`、`fr-FR`、 `es-ES`、`it-IT`、`pt-BR`、`pl-PL`、`tr-TR`、`ja-JP`、`ko-KR`、`zh-CN`、または`zh-TW`。 |
| **Timezone**         | システムのタイムゾーン、または利用できない場合は `UTC` | サポートされている IANA タイムゾーンを選択します。                                                                                                                                                                 |

フィンガープリント シード、ロケール、およびタイムゾーンが必要です。設定では、数字以外のものが含まれるシード、有効な BCP 47 ロケールではないロケール、または有効な IANA タイムゾーンではないタイムゾーンは拒否されます。 GPT-Voice は、これらの値を保存するときに周囲の空白を削除します。

### プロキシ GeoIP はロケールとタイムゾーンを制御します[ネットワーク設定](network.md)]の**GeoIP**でプロキシが有効になっている場合、プロキシはブラウザのロケールとタイムゾーンを決定します。 GPT-Voice は、**Identity** のこれら 2 つのフィールドを無効にし、メッセージ **Proxy GeoIP controls locale and timezone** を表示します。保存されたロケールとタイムゾーンは、後で GeoIP をオフにしても引き続き使用できますが、アクティブなプロキシ GeoIP がそれらを所有している間はブラウザー コンテキストに送信されません。

## 変更を保存する

ブラウザ設定は設定フォームの一部です。値を変更すると、**Unsaved changes** が作成されます。検証が成功したら、**Save changes** を選択します。通常のブラウザ設定は GPT-Voice のローカル設定に保存され、次回 GPT-Voice が該当するブラウザ コンテキストを作成するときに使用されます。保存エラーと破棄確認の動作については、[設定の概要](index.md)]を参照してください。
