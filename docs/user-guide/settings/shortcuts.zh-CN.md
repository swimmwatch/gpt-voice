# 快捷方式设置

GPT-Voice 使用全局快捷方式，因此它们可以在您使用其他应用程序时工作。打开**Settings**并选择**Shortcuts**来查看或更改它们。

| 行动                                  | 默认快捷键 | 当它起作用时                                                               |
| ------------------------------------- | ---------- | -------------------------------------------------------------------------- |
| **Start, pause, or resume recording** | `F9`       | 空闲时开始录制、暂停活动录制或恢复暂停的录制。                             |
| **Stop recording**                    | `F10`      | 停止录音或暂停录音并开始转录。                                             |
| **Cancel**                            | `Escape`   | 取消活动录音。当没有活动记录时，它会取消正在运行的 Prettify 请求。         |
| **Translate selected text**           | `F11`      | 当启用翻译并且​​没有录制或其他选定文本操作处于活动状态时，翻译选定的文本。 |
| **Prettify selected text**            | `F12`      | 当启用美化并且没有录制或其他选定文本操作处于活动状态时，美化选定的文本。   |
| **Retry transcription**               | `Ctrl+F8`  | 仅当 GPT-Voice 空闲时重试最新的可重试转录。                                |

在存在可重试转录之前，重试快捷方式不可用。请参阅[记录和转录](../guides/transcription.md)了解何时可以重试，并参阅[翻译和美化选定文本](../guides/text-actions.md)了解选定文本工作流程。

## 更改快捷方式

1. 选择操作行上的 **Change**。
2. 在捕获对话框中，按您要使用的完整组合键。
3. 检查对话框中显示的组合，然后选择 **Apply**。选择 **Cancel** 使当前快捷方式保持不变。

GPT-Voice 在捕获对话框打开时暂时挂起所有全局快捷方式，然后在您应用或取消时再次注册它们。按非修饰键作为组合的一部分；仅按 `Ctrl`、`Alt`、`Shift` 或平台的命令键不会创建快捷方式。

## 避免冲突

对每个 GPT-Voice 操作使用不同的快捷方式，并选择不与操作系统或其他软件冲突的组合。 GPT-Voice 拒绝冲突的 GPT-Voice 分配。即使其他分配包含修饰符，未修改的键也会与相同的基键冲突，因此请勿配对，例如 F9 与 Ctrl+F9。

如果无法注册新分配，GPT-Voice 会保留当前快捷方式并显示原因。在 macOS 上，捕获的 Command 键表示为 `Command`；在其他支持的平台上，使用等效平台。## 启用或禁用选定文本操作

**Translate** 和 **Prettify** 行各有一个启用开关。两者均默认启用。关闭开关以防止该操作运行，即使按下其快捷键也是如此；打开它以使配置的快捷方式在满足操作的正常条件时再次可用。

这些开关是“设置”表单的一部分，因此在更改它们后选择 **Save changes**。通过捕获对话框更改快捷方式会独立应用该快捷方式；使用设置概述中的未保存更改指示器来区分待处理的表单编辑和已应用的快捷方式。
