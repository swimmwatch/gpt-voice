# Linux

## Linux-пакет deb

Для Ubuntu, Debian, Linux Mint, Pop!_OS та подібних дистрибутивів установіть завантажений пакет:

```bash
sudo apt install ./gpt-voice_*_amd64.deb
```

Якщо ваша система не може встановити локальний пакет deb через `apt`, скористайтеся:

```bash
sudo dpkg -i ./gpt-voice_*_amd64.deb
sudo apt-get install -f
```

Пакет установлює GPT-Voice у `/opt/GPT-Voice`, реєструє ярлик та іконки на робочому столі й надає команду
`gpt-voice`. Запускайте програму з меню застосунків або командою `gpt-voice`.

Щоб оновити програму, установіть новіший пакет deb тією самою командою `apt install`. Щоб видалити пакет,
скористайтеся `sudo apt remove gpt-voice`; використовуйте `sudo apt purge gpt-voice`, якщо також бажаєте вилучити
конфігурацію пакета.

## Linux-пакет rpm

Для Fedora, RHEL, CentOS, openSUSE та подібних дистрибутивів використовуйте менеджер пакетів свого дистрибутива, щоб
він міг розв'язати залежності пакета. Не використовуйте звичайний `rpm -i` для типового встановлення.

```bash
# Fedora, RHEL, CentOS і сумісні дистрибутиви
sudo dnf install ./gpt-voice-*.x86_64.rpm

# Старіші системи CentOS або RHEL
sudo yum install ./gpt-voice-*.x86_64.rpm

# openSUSE
sudo zypper install ./gpt-voice-*.x86_64.rpm
```

Пакет rpm установлює такий самий ярлик, іконки й команду `gpt-voice`, як і пакет deb. Ресурс випуску призначений для
настільних систем `x86_64`. У мінімальному встановленні Linux перед встановленням увімкніть звичайні репозиторії
середовища робочого стола й середовища виконання, щоб менеджер пакетів міг отримати залежності.

Щоб оновити програму, установіть новіший пакет rpm тією самою командою менеджера пакетів. Щоб видалити його,
скористайтеся `sudo dnf remove gpt-voice` у Fedora/RHEL/CentOS або `sudo zypper remove gpt-voice` в openSUSE.

## Linux AppImage

Використовуйте AppImage, якщо бажаєте мати портативну копію замість системного пакета.

1. Завантажте `GPT-Voice-*.AppImage`.
2. Зробіть файл виконуваним і запустіть його:

   ```bash
   chmod +x GPT-Voice-*.AppImage
   ./GPT-Voice-*.AppImage
   ```

Під час першого запуску GPT-Voice за можливості реєструє локальний ярлик та іконку на робочому столі для вашого
користувача. Щоб оновити програму, завантажте новіший AppImage, зробіть його виконуваним і запустіть замість старого
файлу.

Щоб видалити встановлення AppImage, завершіть GPT-Voice, запустіть з цього AppImage команду видалення інтеграції з
робочим столом, а потім видаліть файл:

```bash
./GPT-Voice-*.AppImage --remove-linux-appimage-desktop-integration
```

## Збережені дані Linux

Видалення пакета deb або rpm чи видалення AppImage не прибирає ваші налаштування та збережений сеанс постачальника.
Вони залишаються в `~/.config/GPT-Voice`. Видаляйте цей каталог вручну лише тоді, коли хочете виконати чисте скидання.

Після завершення встановлення перейдіть до [першого використання](../getting-started.md), щоб підключити постачальника
розшифрування й зробити перший запис.
