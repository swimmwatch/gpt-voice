# Linux

## लिनक्स डिबेट पैकेज

उबंटू, डेबियन, लिनक्स मिंट, पॉप!_ओएस और इसी तरह के वितरण के लिए, डाउनलोड किया गया पैकेज इंस्टॉल करें:

```bash
sudo apt install ./gpt-voice_*_amd64.deb
```

यदि आपका सिस्टम `apt` के माध्यम से स्थानीय डिबेट स्थापित नहीं कर सकता है, तो इसका उपयोग करें:

```bash
sudo dpkg -i ./gpt-voice_*_amd64.deb
sudo apt-get install -f
```

पैकेज `/opt/GPT-Voice` में GPT-Voice स्थापित करता है, एक डेस्कटॉप लॉन्चर और आइकन पंजीकृत करता है, और प्रदान करता है
`gpt-voice` कमांड। इसे अपने एप्लिकेशन मेनू से या `gpt-voice` से लॉन्च करें।

अपडेट करने के लिए, उसी `apt install` कमांड के साथ नया डिब इंस्टॉल करें। पैकेज को हटाने के लिए, उपयोग करें`sudo apt remove gpt-voice`; यदि आप भी पैकेज कॉन्फ़िगरेशन हटाना चाहते हैं तो `sudo apt purge gpt-voice` का उपयोग करें।

## लिनक्स आरपीएम पैकेज

फेडोरा, आरएचईएल, सेंटओएस, ओपनएसयूएसई और समान वितरण के लिए, अपने वितरण पैकेज प्रबंधक का उपयोग करें ताकि यह हल हो सके
पैकेज निर्भरताएँ। सामान्य इंस्टालेशन के लिए सादे `rpm -i` का उपयोग न करें।

```bash
# Fedora, RHEL, CentOS, and compatible distributions
sudo dnf install ./gpt-voice-*.x86_64.rpm

# Older CentOS or RHEL systems
sudo yum install ./gpt-voice-*.x86_64.rpm

# openSUSE
sudo zypper install ./gpt-voice-*.x86_64.rpm
```

आरपीएम पैकेज डिबेट पैकेज के समान लॉन्चर, आइकन और `gpt-voice` कमांड स्थापित करता है। इसकी रिलीज़ एसेट है
`x86_64` डेस्कटॉप सिस्टम के लिए। न्यूनतम लिनक्स इंस्टॉलेशन पर, पहले सामान्य डेस्कटॉप/रनटाइम रिपॉजिटरी सक्षम करें
स्थापित करना ताकि आपका पैकेज प्रबंधक इसकी निर्भरता प्राप्त कर सके।

अद्यतन करने के लिए, उसी पैकेज-मैनेजर कमांड के साथ नया आरपीएम स्थापित करें। इसे दूर करने के लिए उपयोग करें
`sudo dnf remove gpt-voice` Fedora/RHEL/CentOS पर या `sudo zypper remove gpt-voice` openSUSE पर।

## लिनक्स ऐपइमेज

जब आप सिस्टम पैकेज के बजाय पोर्टेबल कॉपी पसंद करते हैं तो AppImage का उपयोग करें।

1. डाउनलोड करें `GPT-Voice-*.AppImage`।
2. इसे निष्पादन योग्य बनाएं और चलाएं:

   ```bash
   chmod +x GPT-Voice-*.AppImage
   ./GPT-Voice-*.AppImage
   ```

अपने पहले लॉन्च पर, GPT-Voice जब संभव हो तो आपके उपयोगकर्ता के लिए एक स्थानीय डेस्कटॉप लॉन्चर और आइकन पंजीकृत करता है। अद्यतन करने के लिए,
नया AppImage डाउनलोड करें, इसे निष्पादन योग्य बनाएं, और पुरानी फ़ाइल के बजाय इसे चलाएँ।

किसी AppImage इंस्टॉलेशन को हटाने के लिए, GPT-Voice को छोड़ें, उस AppImage से डेस्कटॉप-इंटीग्रेशन रिमूवल कमांड चलाएँ,
फिर फ़ाइल हटाएँ:

```bash
./GPT-Voice-*.AppImage --remove-linux-appimage-desktop-integration
```

## बनाए रखा लिनक्स डेटा

डिबेट या आरपीएम पैकेज को हटाने या ऐपइमेज को हटाने से आपकी सेटिंग्स या सहेजा गया प्रदाता सत्र नहीं हटता है। वे
`~/.config/GPT-Voice` में बने रहें। उस निर्देशिका को मैन्युअल रूप से तभी हटाएं जब आप क्लीन रीसेट चाहते हों।

जब इंस्टॉलेशन पूरा हो जाए, तो ट्रांसक्रिप्शन प्रदाता को कनेक्ट करने के लिए [पहली बार ](../getting-started.md) का उपयोग जारी रखें और
अपनी पहली रिकॉर्डिंग करें.
