# Частые вопросы

## GPT-Voice расшифровывает речь сам по себе?

Нет. GPT-Voice — настольное приложение, которое отправляет запись выбранному вами поставщику расшифровки: ChatGPT
Web через сеанс браузера, в который вы вошли, либо OpenAI API через ваш ключ API. Доступность поставщика, оплата,
квоты и условия определяются этим поставщиком. См. [выбор поставщика расшифровки и управление им](guides/providers.md).

## Что покидает мой компьютер?

После остановки записи подготовленное аудио отправляется выбранному поставщику расшифровки. Translate отправляет
выделенный текст в Google Translate. Prettify отправляет выделенный текст и настроенный запрос Prettify на конечную
точку Ollama или vLLM. Полное описание потоков данных и локального хранения приведено в разделе
[конфиденциальность и данные](privacy.md).

## GPT-Voice вставляет результат в моё приложение?

Нет. Успешные результаты расшифровки, Translate и Prettify копируются в системный буфер обмена. Вставьте результат
туда, где он нужен. См. [запись и расшифровка речи](guides/transcription.md) и
[перевод и Prettify для выделенного текста](guides/text-actions.md).

## Можно ли использовать GPT-Voice без ключа OpenAI API?

Да. ChatGPT Web использует сеанс браузера ChatGPT, в который вы вошли, а не ключ API. Это отдельный от OpenAI API
поставщик с собственными требованиями к учётной записи. См. [настройки поставщика](settings/providers.md).

## Может ли GPT-Voice работать полностью офлайн?

Не для расшифровки или Translate: эти функции используют выбранную удалённую службу. Prettify может использовать
локальную конечную точку Ollama или vLLM, когда вы запустили эту службу на том же компьютере, однако GPT-Voice не
устанавливает и не управляет этой конечной точкой. См. [настройки Prettify](settings/prettify.md).

## Какие платформы поддерживаются?

Текущие выпуски поддерживают Windows и Linux через установщик Windows, а также пакеты deb, rpm и AppImage. Выпуски
для macOS приостановлены, пока готовятся подписание и нотариализация. См. [установка, обновление или удаление](install.md).

## Удаляет ли обновление или удаление приложения мои настройки?

Нет. Обычные способы удаления намеренно сохраняют локальные данные приложения, включая настройки и сохранённые данные
поставщиков. Если нужно целенаправленно сбросить эти данные, воспользуйтесь инструкциями по удалению в разделе
[конфиденциальность и данные](privacy.md).

## Почему не сработало сочетание клавиш или действие с выделенным текстом?

Проверьте, что действие включено, его сочетание сохранено и другая программа не зарезервировала ту же комбинацию
клавиш. Translate и Prettify выполняются по одному и ждут окончания любой активной записи. См.
[настройки сочетаний клавиш](settings/shortcuts.md) и [устранение неполадок](troubleshooting.md).

## Можно ли использовать прокси?

Да. GPT-Voice может передавать HTTP-, HTTPS- или SOCKS5-прокси своим браузерным контекстам. Учётные данные для SOCKS5
не поддерживаются, а Proxy GeoIP может управлять Locale и Timezone браузера. См.
[настройки Network](settings/network.md).

## Как очистить расшифровку, сеанс или ключ?

Используйте **Clear history** в окне History для сохранённых расшифровок. Используйте соответствующий элемент очистки
поставщика для сеанса ChatGPT, ключа OpenAI API, ключа vLLM или пароля прокси. Точная область действия каждого сброса
описана в разделе [конфиденциальность и данные](privacy.md).
