# История расшифровок и системный трей

GPT-Voice хранит локальную историю успешных расшифровок, чтобы вы могли снова использовать результат после того, как
он исчез из буфера обмена. Трей предоставляет доступ к приложению, когда его главное окно скрыто.

## Повторное использование истории расшифровок

Откройте меню трея и выберите **History**. Каждая успешная расшифровка сохраняется локально со временем запроса,
именем поставщика и текстом. История хранится в локальных данных SQLite приложения; записанное аудио в ней не
хранится. Записи могут содержать конфиденциальный продиктованный текст, поэтому относитесь к истории как к любому
другому локальному документу.

Новые записи показываются первыми. История подгружается постепенно при прокрутке, поэтому длинную историю не нужно
загружать одним запросом. Окно показывает сообщения о загрузке, повторной попытке и безопасные сообщения об ошибках,
если не удаётся получить следующую страницу.

Чтобы снова использовать запись, выберите её карточку текста. GPT-Voice копирует сохранённый текст этой записи в
системный буфер обмена и ненадолго показывает **Copied**. Текст не отправляется на расшифровку повторно. Если
копирование не удалось или запись уже удалена, окно истории сообщает о сбое вместо изменения буфера обмена.

## Очистите локальную историю

Используйте **Clear history** в окне History и подтвердите диалог, чтобы удалить все сохранённые записи расшифровок.
Это действие очищает локальную историю и не может быть отменено в GPT-Voice. После этого новые успешные расшифровки
снова создают записи.

Если вы хотите удалить все сохранённые данные приложения при удалении GPT-Voice, следуйте инструкциям для своей
платформы в разделе [установка, обновление или удаление](../install.md).

## Использование трея

Закрытие главного окна GPT-Voice скрывает его, а не завершает работу приложения. Оно продолжает работать в системном
трее, поэтому настроенные глобальные сочетания клавиш остаются доступными. Выберите значок в трее, чтобы перевести
фокус на видимое главное окно или показать его, если оно скрыто.

Меню трея содержит следующие действия:

| Действие меню      | Результат                                                                                       |
| ------------------ | ----------------------------------------------------------------------------------------------- |
| **Show GPT-Voice** | Показывает главное окно и переводит на него фокус либо создаёт его при необходимости.           |
| **Settings**       | Открывает окно Settings.                                                                        |
| **History**        | Открывает локальную историю расшифровок.                                                        |
| **About**          | Открывает окно About.                                                                           |
| **Quit**           | Завершает GPT-Voice. Используйте это действие, чтобы остановить, а не просто скрыть приложение. |

Значок в трее отражает текущую активность: бездействие, запись, пауза, обработка расшифровки или Prettify. Это
индикатор и точка навигации; элементы управления записью остаются доступными через панель команд и настроенные
сочетания клавиш.
