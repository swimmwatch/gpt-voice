# 转录历史和托盘

GPT-Voice 保留成功转录的本地历史记录，以便您可以在结果离开剪贴板后重复使用结果。
当应用程序的主窗口隐藏时，托盘提供对应用程序的访问。

## 重用转录历史记录

打开托盘菜单并选择 **History**。每个成功的转录都及其请求时间存储在本地，
提供者名称和文本。历史记录存储在应用程序的本地 SQLite 数据中；它不存储记录的
音频。由于条目可能包含敏感的听写文本，因此请像对待任何其他本地文档一样对待历史记录。

最新的条目首先出现。当您滚动时，历史记录会逐渐加载，因此不需要加载很长的历史记录
一个请求。如果无法获取下一页，该窗口将显示加载、重试和安全错误消息。

要重复使用条目，请选择其文本卡。 GPT-Voice 将该条目的存储文本复制到系统剪贴板并短暂
显示 **Copied**。它不会再次提交文本进行转录。如果复制失败或条目已被复制
删除后，历史记录窗口会报告失败而不是更改剪贴板。

## 清除本地历史记录

使用“历史记录”窗口中的 **Clear history** 并确认对话框以删除每个保存的转录条目。这个
行动清除了当地的历史；它无法从 GPT-Voice 撤消。新的成功转录创建新条目
之后。

如果您想在卸载 GPT-Voice 过程中删除所有保留的应用程序数据，请按照特定于平台的说明进行操作
[安装、更新或删除](../install.md)中的说明。

## 使用托盘

关闭主 GPT-Voice 窗口会将其隐藏，而不是退出应用程序。应用程序继续运行在
系统托盘，因此其配置的全局快捷方式仍然可用。选择托盘图标以聚焦可见的主窗口
或者如果它被隐藏则显示它。

托盘菜单提供以下操作：

| 菜单操作           | 结果                                                                       |
| ------------------ | -------------------------------------------------------------------------- |
| **Show GPT-Voice** | 显示并聚焦主窗口，或者在必要时创建它。                                     |
| **Settings**       | 打开“设置”窗口。                                                           |
| **History**        | 打开本地转录历史记录。                                                     |
| **About**          | 打开“关于”窗口。                                                           |
| **Quit**           | 退出 GPT-Voice。当您想要停止应用程序而不仅仅是隐藏其窗口时，请使用此选项。 |

托盘图标反映当前活动：空闲、录制、暂停、转录处理或美化。它是一个指示器和导航点；录音控制仍然可以通过 Command Dock 及其配置
快捷方式。
