# 転写履歴とトレイ

GPT-Voice は、成功した文字起こしのローカル履歴を保持するため、結果がクリップボードから出た後に再利用できます。
メイン ウィンドウが非表示になっている場合、トレイを使用してアプリケーションにアクセスできます。

## 文字起こし履歴を再利用する

トレイメニューを開き、**History**を選択します。成功した各転写は、リクエスト時刻とともにローカルに保存されます。

## ローカル履歴をクリアする

[履歴] ウィンドウで **Clear history** を使用し、ダイアログを確認して、保存されているすべての文字起こしエントリを削除します。これ
アクションによりローカル履歴がクリアされます。 GPT-Voice から元に戻すことはできません。新たに成功した文字起こしにより、新しいエントリが作成されます
その後。

GPT-Voice のアンインストールの一環として、保持されているすべてのアプリケーション データを削除する場合は、プラットフォーム固有の手順に従ってください。
[インストール、アップデート、または削除](../install.md)の手順を参照してください。

## トレイを使用する

アプリケーションを終了するのではなく、メイン GPT-Voice ウィンドウを閉じると非表示になります。アプリケーションは引き続き実行されます
システム トレイに保存されるため、設定されたグローバル ショートカットは利用可能なままになります。トレイ アイコンを選択して、表示されているメイン ウィンドウにフォーカスします。
または、非表示の場合は表示します。

トレイ メニューでは次のアクションが提供されます。

| メニューアクション | 結果                                                                                                                                   |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| **Show GPT-Voice** | メイン ウィンドウを表示してフォーカスするか、必要に応じて作成します。                                                                  |
| **Settings**       | 設定ウィンドウを開きます。                                                                                                             |
| **History**        | ローカルの文字起こし履歴を開きます。                                                                                                   |
| **About**          | 「バージョン情報」ウィンドウを開きます。                                                                                               |
| **Quit**           | GPT-Voice を終了します。単にアプリケーションのウィンドウを非表示にするのではなく、アプリケーションを停止したい場合にこれを使用します。 |

トレイ アイコンは、現在のアクティビティ (アイドル、録音、一時停止、文字起こし処理、または Prettify) を反映します。それはインジケーターとナビゲーションポイント。録音コントロールは、Command Dock およびその設定を通じて引き続き利用可能です
ショートカット。
