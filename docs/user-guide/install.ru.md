# Установка, обновление или удаление GPT-Voice

Для GPT-Voice доступны готовые к запуску выпуски для Windows и Linux. Не требуется отдельно устанавливать Node.js,
npm, локальную модель, CUDA или Whisper.

Скачайте пакет для своего компьютера из [GitHub Releases](https://github.com/swimmwatch/gpt-voice/releases). Каждый
выпуск также содержит файлы `SHA256SUMS-*.txt` для соответствующих платформ. Чтобы проверить загрузку, сравните
значение SHA-256, полученное штатной утилитой контрольных сумм вашей операционной системы, со строкой для скачанного
файла выпуска.

> Выпуски для macOS приостановлены, пока готовятся подпись Developer ID и нотарификация. Не устанавливайте
> неофициальный DMG под видом GPT-Voice: в текущих выпусках нет поддерживаемого пакета для macOS.

## Выберите файл выпуска

| Платформа              | Файл выпуска             | Когда использовать                                                 |
| ---------------------- | ------------------------ | ------------------------------------------------------------------ |
| Windows                | `GPT-Voice Setup *.exe`  | Для обычной установки в Windows.                                   |
| Linux семейства Debian | `gpt-voice_*_amd64.deb`  | Для Ubuntu, Debian, Linux Mint, Pop!_OS или похожего дистрибутива. |
| Linux семейства RPM    | `gpt-voice-*.x86_64.rpm` | Для Fedora, RHEL, CentOS, openSUSE или похожего дистрибутива.      |
| Linux                  | `GPT-Voice-*.AppImage`   | Для переносимой версии без установки системного пакета.            |
