# Установка, обновление или удаление GPT-Voice

Для GPT-Voice доступны готовые к запуску выпуски для Windows и Linux. Не требуется отдельно устанавливать Node.js,
npm, локальную модель, CUDA или Whisper.

Скачайте пакет для своего компьютера из [GitHub Releases](https://github.com/swimmwatch/gpt-voice/releases). Каждый
выпуск также содержит файлы `SHA256SUMS-*.txt` для соответствующих платформ. Чтобы проверить загрузку, сравните
значение SHA-256, полученное штатной утилитой контрольных сумм вашей операционной системы, со строкой для скачанного
файла выпуска.

> Выпуски для macOS приостановлены, пока готовятся подпись Developer ID и нотарификация. Не устанавливайте
> неофициальный DMG под видом GPT-Voice: в текущих выпусках нет поддерживаемого пакета для macOS.

## Выберите файл выпуска

| Платформа              | Файл выпуска             | Когда использовать                                                 |
| ---------------------- | ------------------------ | ------------------------------------------------------------------ |
| Windows                | `GPT-Voice Setup *.exe`  | Для обычной установки в Windows.                                   |
| Linux семейства Debian | `gpt-voice_*_amd64.deb`  | Для Ubuntu, Debian, Linux Mint, Pop!_OS или похожего дистрибутива. |
| Linux семейства RPM    | `gpt-voice-*.x86_64.rpm` | Для Fedora, RHEL, CentOS, openSUSE или похожего дистрибутива.      |
| Linux                  | `GPT-Voice-*.AppImage`   | Для переносимой версии без установки системного пакета.            |

## Windows

1. Скачайте `GPT-Voice Setup *.exe` из последнего выпуска и откройте его.
2. При необходимости выберите место установки.
3. Оставьте включёнными ярлыки на рабочем столе и в меню «Пуск», если не хотите запускать приложение вручную.
4. Завершите работу установщика, затем откройте **GPT-Voice** из меню «Пуск», с ярлыка на рабочем столе или на
   последнем экране установщика.

Пакет Windows — установщик NSIS. Он устанавливает GPT-Voice, встроенную среду браузера, значки, ярлыки и запись
деинсталлятора в настройках Windows.

### Обновление или удаление в Windows

Чтобы обновить приложение, скачайте более новый `GPT-Voice Setup *.exe` и запустите его поверх существующей установки.

Чтобы удалить приложение:

1. Откройте **Settings** > **Apps** > **Installed apps**.
2. Найдите **GPT-Voice**.
3. Выберите **Uninstall**.

Удаление приложения удаляет установленные файлы и ярлыки, но намеренно сохраняет локальные настройки и сохранённый
сеанс поставщика в `%APPDATA%\GPT-Voice`. Поэтому после переустановки их можно использовать снова. Удаляйте эту папку
вручную, только если хотите удалить и эти локальные данные.

## Пакет Linux deb

Для Ubuntu, Debian, Linux Mint, Pop!_OS и похожих дистрибутивов установите скачанный пакет:

```bash
sudo apt install ./gpt-voice_*_amd64.deb
```

Если система не может установить локальный deb через `apt`, используйте:

```bash
sudo dpkg -i ./gpt-voice_*_amd64.deb
sudo apt-get install -f
```

Пакет устанавливает GPT-Voice в `/opt/GPT-Voice`, регистрирует ярлык и значки рабочего стола и предоставляет команду
`gpt-voice`. Запускайте приложение из меню приложений или командой `gpt-voice`.

Чтобы обновить приложение, установите более новый deb той же командой `apt install`. Чтобы удалить пакет, используйте
`sudo apt remove gpt-voice`; используйте `sudo apt purge gpt-voice`, если также хотите удалить конфигурацию пакета.

## Пакет Linux rpm

Для Fedora, RHEL, CentOS, openSUSE и похожих дистрибутивов используйте менеджер пакетов вашего дистрибутива, чтобы он
мог разрешить зависимости пакета. Не используйте для обычной установки простой `rpm -i`.

```bash
# Fedora, RHEL, CentOS и совместимые системы
sudo dnf install ./gpt-voice-*.x86_64.rpm

# Более старые CentOS или RHEL
sudo yum install ./gpt-voice-*.x86_64.rpm

# openSUSE
sudo zypper install ./gpt-voice-*.x86_64.rpm
```

Пакет rpm устанавливает тот же ярлык, значки и команду `gpt-voice`, что и deb. Его файл выпуска предназначен для
настольных систем `x86_64`. На минимальной установке Linux включите обычные репозитории среды рабочего стола до
установки, чтобы менеджер пакетов мог получить зависимости.

Чтобы обновить приложение, установите более новый rpm той же командой менеджера пакетов. Чтобы удалить его,
используйте `sudo dnf remove gpt-voice` в Fedora/RHEL/CentOS или `sudo zypper remove gpt-voice` в openSUSE.

## Linux AppImage

Используйте AppImage, если предпочитаете переносимую копию системному пакету.

1. Скачайте `GPT-Voice-*.AppImage`.
2. Сделайте файл исполняемым и запустите его:

   ```bash
   chmod +x GPT-Voice-*.AppImage
   ./GPT-Voice-*.AppImage
   ```

При первом запуске GPT-Voice по возможности регистрирует локальный ярлык и значок рабочего стола для вашего
пользователя. Для обновления скачайте более новый AppImage, сделайте его исполняемым и запустите вместо старого файла.

Чтобы удалить установку AppImage, завершите GPT-Voice, выполните из этого AppImage команду удаления интеграции с
рабочим столом, затем удалите файл:

```bash
./GPT-Voice-*.AppImage --remove-linux-appimage-desktop-integration
```

## Сохранённые данные Linux

Удаление пакета deb или rpm либо удаление AppImage не удаляет настройки и сохранённый сеанс поставщика. Они остаются в
`~/.config/GPT-Voice`. Удаляйте этот каталог вручную, только если хотите полностью сбросить данные.

После установки перейдите к [первому запуску](getting-started.md), чтобы подключить поставщика расшифровки и сделать
первую запись.
