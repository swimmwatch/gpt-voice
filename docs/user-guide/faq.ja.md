# よくある質問

## GPT-Voice は音声を単独で転写しますか?

いいえ。 GPT-Voice は、選択した文字起こしプロバイダーに録音を送信するデスクトップ アプリケーションです: ChatGPT Web
サインインしているブラウザー セッションを通じて、または OpenAI API 独自の API キーを通じて。プロバイダーの可用性、請求、
クォータと条件はそのプロバイダーによって管理されます。 [文字起こしプロバイダーの選択と管理](guides/providers.md)を参照してください。

## 私のコンピューターに何が残ったのでしょうか?

録音を停止すると、準備された音声が選択した文字起こしプロバイダーに送信されます。選択したテキストを翻訳により送信されます
Google Translate。 Prettify は、選択されたテキストとその構成されたプロンプトを Ollama または vLLM エンドポイントに送信します。参照
完全なデータフローとローカル保持の詳細については、[privacy and data](privacy.md)を参照してください。

## GPT-Voice は結果をアプリケーションに入力しますか?

いいえ。成功した文字起こし、翻訳、Prettify の結果はシステムのクリップボードにコピーされます。結果を貼り付けます
必要な場所に。 [記録と転写](guides/transcription.md)および
[選択したテキストを翻訳して整形します](guides/text-actions.md)。

## OpenAI API キーなしで GPT-Voice を使用できますか?

はい。 ChatGPT Web は、API キーの代わりに、サインインした ChatGPT ブラウザー セッションを使用します。 OpenAI APIとは別のものです
プロバイダーとそのアカウントの要件。 [プロバイダー設定](settings/providers.md)を参照してください。

## GPT-Voice は完全にオフラインで動作できますか?

文字起こしや翻訳には使用できません。これらの機能は、選択したリモート サービスを使用します。 Prettify はローカルを使用できます
Ollama または vLLM エンドポイントを同じコンピューター上でサービスを実行する場合、GPT-Voice は
あなたのためのエンドポイント。 [Prettify設定](settings/prettify.md)を参照してください。

## どのプラットフォームがサポートされていますか?

現在のリリースは、Windows インストーラー、deb、rpm、および AppImage パッケージを通じて Windows と Linux をサポートしています。 macOS
署名と公証の準備が行われている間、リリースは一時停止されます。 [インストール、更新、または削除](install.md)を参照してください。

## アップデートまたはアンインストールすると設定が消去されますか?

いいえ。通常のアンインストール パスでは、設定や保存されたプロバイダー データなどのローカル アプリケーション データが意図的に保持されます。
データを意図的にリセットする場合は、[プライバシーとデータ](privacy.md)の削除手順を使用してください。

## ショートカットまたは選択したテキストのアクションが実行されなかったのはなぜですか?

アクションが有効になっていること、そのショートカットが保存されていること、および別のアプリケーションが同じキーを予約していないことを確認します。
組み合わせ。 Translation と Prettify は一度に 1 つずつ実行され、録音が終了するまで待機します。参照
[ショートカット設定](settings/shortcuts.md)および[トラブルシューティング](troubleshooting.md)。

## プロキシを使用できますか?

はい。 GPT-Voice は、HTTP、HTTPS、または SOCKS5 プロキシをブラウザー コンテキストに渡すことができます。 SOCKS5 認証情報はサポートされていません。
プロキシ GeoIP はブラウザーのロケールとタイムゾーンを制御できます。 [ネットワーク設定](settings/network.md)を参照してください。
