#अक्सर पूछे जाने वाले प्रश्न

## क्या GPT-Voice भाषण को स्वयं ट्रांसक्रिप्ट करता है?

नहीं, GPT-Voice एक डेस्कटॉप एप्लिकेशन है जो आपके द्वारा चुने गए ट्रांसक्रिप्शन प्रदाता को रिकॉर्डिंग भेजता है: ChatGPT Web
आपके साइन-इन ब्राउज़र सत्र के माध्यम से, या अपनी स्वयं की एपीआई कुंजी के माध्यम से OpenAI API। प्रदाता उपलब्धता, बिलिंग,
कोटा और शर्तें उस प्रदाता द्वारा नियंत्रित होती हैं। देखें [एक प्रतिलेखन प्रदाता चुनें और प्रबंधित करें](guides/providers.md)।

## मेरा कंप्यूटर क्या छोड़ता है?

रिकॉर्डिंग बंद करने से तैयार ऑडियो चयनित ट्रांसक्रिप्शन प्रदाता को भेज दिया जाता है। अनुवाद चयनित पाठ को भेजता है
Google Translate. Prettify चयनित टेक्स्ट और उसके कॉन्फ़िगर किए गए प्रॉम्प्ट को आपके Ollama या vLLM एंडपॉइंट पर भेजता है। देखिये
[संपूर्ण डेटा-प्रवाह और स्थानीय-अवधारण विवरण के लिए गोपनीयता और डेटा](privacy.md)।

## क्या GPT-Voice मेरे एप्लिकेशन में परिणाम टाइप करता है?

नहीं, सफल प्रतिलेखन, अनुवाद और प्रीटिफाई परिणाम सिस्टम क्लिपबोर्ड पर कॉपी किए जाते हैं। परिणाम चिपकाएँ
जहां आपको इसकी आवश्यकता है. देखें [रिकॉर्ड और ट्रांसक्राइब](guides/transcription.md) और
[चयनित टेक्स्ट का अनुवाद और सुंदरीकरण करें](guides/text-actions.md)।

## क्या मैं OpenAI API कुंजी के बिना GPT-Voice का उपयोग कर सकता हूं?

हाँ। ChatGPT Web एपीआई कुंजी के बजाय साइन-इन किए हुए ChatGPT Web ब्राउज़र सत्र का उपयोग करता है। यह OpenAI API से अलग है
प्रदाता और उसकी खाता आवश्यकताएँ। [प्रदाता सेटिंग्स](settings/providers.md) देखें।

## क्या GPT-Voice पूरी तरह ऑफ़लाइन काम कर सकता है?

प्रतिलेखन या अनुवाद के लिए नहीं: वे सुविधाएँ उनकी चयनित दूरस्थ सेवा का उपयोग करती हैं। Prettify स्थानीय का उपयोग कर सकता है
Ollama या vLLM एंडपॉइंट जब आप उस सेवा को उसी कंप्यूटर पर चलाते हैं, लेकिन GPT-Voice स्थापित या संचालित नहीं करता है
आपके लिए समापन बिंदु. [सुंदर सेटिंग्स](settings/prettify.md) देखें।

## कौन से प्लेटफ़ॉर्म समर्थित हैं?

वर्तमान रिलीज़ विंडोज़ इंस्टॉलर, डिबेट, आरपीएम और ऐपइमेज पैकेज के माध्यम से विंडोज़ और लिनक्स का समर्थन करते हैं। macOS
हस्ताक्षर और नोटरीकरण तैयार करते समय रिलीज़ रोक दी जाती है। देखें [इंस्टॉल करें, अपडेट करें या निकालें](install.md)।

## क्या कोई अपडेट या अनइंस्टॉल मेरी सेटिंग्स मिटा देता है?

नहीं, सामान्य अनइंस्टॉल पथ जानबूझकर स्थानीय एप्लिकेशन डेटा को बनाए रखते हैं, जिसमें सेटिंग्स और सहेजे गए प्रदाता डेटा भी शामिल हैं।
जब आप जानबूझकर उस डेटा को रीसेट करना चाहते हैं तो [गोपनीयता और डेटा](privacy.md) में निष्कासन निर्देशों का उपयोग करें।

## मेरी शॉर्टकट या चयनित-पाठ क्रिया क्यों नहीं चली?

पुष्टि करें कि कार्रवाई सक्षम है, इसका शॉर्टकट सहेजा गया है, और किसी अन्य एप्लिकेशन ने समान कुंजी आरक्षित नहीं की है
संयोजन. अनुवाद और Prettify एक समय में एक चलाते हैं और किसी भी रिकॉर्डिंग के समाप्त होने तक प्रतीक्षा करते हैं। देखिये
[शॉर्टकट सेटिंग्स](settings/shortcuts.md) और [समस्या निवारण](troubleshooting.md)।

## क्या मैं प्रॉक्सी का उपयोग कर सकता हूँ?

हाँ। GPT-Voice अपने ब्राउज़र संदर्भों में HTTP, HTTPS, या SOCKS5 प्रॉक्सी पास कर सकता है। SOCKS5 क्रेडेंशियल समर्थित नहीं हैं,
और प्रॉक्सी GeoIP ब्राउज़र लोकेल और टाइमज़ोन का नियंत्रण ले सकता है। [नेटवर्क सेटिंग्स](settings/network.md) देखें।

## मैं किसी प्रतिलेख, सत्र या कुंजी को कैसे साफ़ करूँ?

सहेजे गए ट्रांसक्रिप्शन के लिए इतिहास विंडो में **Clear history** का उपयोग करें। इसके लिए संबंधित प्रदाता के स्पष्ट नियंत्रण का उपयोग करें
ChatGPT Web सत्र, OpenAI API कुंजी, vLLM कुंजी, या प्रॉक्सी पासवर्ड। सटीक दायरे के लिए [गोपनीयता और डेटा](privacy.md) देखें
प्रत्येक रीसेट का.
