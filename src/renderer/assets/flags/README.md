# Language Flag Assets

The language selector uses public-domain SVG flag assets from Wikimedia Commons:

- `en.svg`: [Flag of the United States](https://commons.wikimedia.org/wiki/File:Flag_of_the_United_States.svg)
- `ru.svg`: [Flag of Russia](https://commons.wikimedia.org/wiki/File:Flag_of_Russia.svg)
- `uk.svg`: [Flag of Ukraine](https://commons.wikimedia.org/wiki/File:Flag_of_Ukraine.svg)
- `be.svg`: [Flag of Belarus](https://commons.wikimedia.org/wiki/File:Flag_of_Belarus.svg)

Keep the files local so the Electron renderer does not make network requests for interface assets.
